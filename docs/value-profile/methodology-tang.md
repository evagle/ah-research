## §A. 唐朝 分析 框架 总览

### A.1 核心 mantra (最高抽象, 贯穿所有分析)

> "**企业的内在价值是企业未来所有自由现金流的折现值之和**" — 格雷厄姆 → 巴菲特 → 老唐 继承

这句话在老唐文集里反复出现。它不是装饰语。它决定了所有后续动作的取舍:
- 不看 "市值 / 股价 / 热度"，只看 "自由现金流的可持续性"。
- 看 PE 时实际看的是 "投资回报率的倒数" (vol2 L2970)。同样期限的债券因还本付息确定性接近, 看 PE 足矣；两家企业之间的 PE 比较"没有意义"，除非先解决三大前提。

### A.2 三大前提 (估值大厦的承重墙) ★★★★★

任何进入估值环节之前，**必须**先对这三个问题给出 "Yes/No + 证据":

```
① 净利润数据是否为真?            (True earnings test)
② 未来能否持续?                   (Durability test)
③ 持续是否需要依赖加大资本投入?   (Free cash flow vs. reinvestment test)
```

出处: vol1 L4309-4315 (2017 茅台 估值文)；vol2 L2980-3001 (长安汽车 对比茅台, 经典阐述)。

**关键应用细节**:
- 三前提 **必须全部满足**。任何一项成 "No"，老唐的 25x / 合理估值法不适用，必须打折或放弃。
- "持续是否需要加大资本投入" = **自由现金流** 的核心。老唐对 "自由现金流" 的定义 (vol2 L6856): **经营所得 − 维持现有盈利能力所需的资本支出**。这不是一般的 "经营现金流 − CapEx"，而是 "只减去 **维持性** CapEx"。
- 典型案例 (vol2 L2988-3001): 甲乙两家公司今年都赚 10 亿。甲能全部分光; 乙只能分一半，另一半必须再投入才能维持明年的 10 亿。同价格，必选甲，因其自由现金流更高。长安汽车报表 PE 7 倍，但为维持销量每年需投 65 亿 CapEx，这 65 亿是 "泡沫利润"，需要 "去泡沫化"。

### A.3 三好 标准 (好生意 / 好公司 / 好价格)

出处: vol1 L12394 (古井贡 分析, 总结为 "一般一般"); vol1 L14663-14665 (雅戈尔 "好公司" 三维度).

| 层级 | 判断对象 | 老唐的检验问题 |
|---|---|---|
| 好生意 | **行业 / 商业模式** | 盈利是否 "真金白银" (高毛利 + 低应收 + 低 CapEx)？客户为什么 必须 买？定价权在哪里？ |
| 好公司 | **管理层 + 护城河** | 管理层历史言行一致吗？分红率 / 股权激励 / 关联交易 是否对小股东友好？ROE 是否持续高位？ |
| 好价格 | **估值 / 市场位置** | 三年后合理估值打 5 折 (非高杠杆) 或 7 折 (高杠杆) 以下? |

三者**顺序不可交换**: 先确认好生意, 再看好公司, 最后算好价格。老唐反复强调 "否决一个就放弃, 不和稀泥"。

### A.4 研究 步骤 (实战流程)

从 vol1 福寿园 "如何从零开始研究一家企业" 系列 (L342-346) 与 雅戈尔 《生人勿近诞生记》(vol2 L15305-15306 "列为必读") 归纳:

1. **招股说明书 + 历年财报** — "首选，不仅权威而且免费" (vol1 L5458)。先读最老一份招股说明书, 看原始产能 / 初始资本 / 最初的 "讲故事" 与现实差异。
2. **资产负债表 简化** — 把资产压缩成 "类现金 / 经营资产 / 生产资产 / 投资资产 / 商誉与无形"。vol2 L10709, L11256, L12912 反复演示 (信立泰): **类现金 61 亿 = 货币 + 银票 + 预付 + 其他**。如果总市值 ≈ 类现金 + 欠条盈余，那市场在白送生意。
3. **利润表 + 现金流量表 交叉验证** — 销售收现 vs. 营业收入 比值, 经营现金流 vs. 净利润 比值, 逐年对比。详见 §D.3.
4. **"去掉财技"** 还原真实销售 — 茅台案例: 报表营收 + (期末预收 − 期初预收)/1.17 (vol1 L1573)。
5. **形成 "三年后合理估值"** — 老唐估值法 (§F).
6. **设买点 + 卖点, 然后收工睡觉** — vol1 L6972: "低于 9500 亿买入, 高于 23000 亿卖出。市值波动于 9500 亿至 23000 亿之间, 持股不动。收工睡觉。简单吧?"

### A.5 心理 / 行为 层 (常被忽略但必讲)

- "**牛市只是投资者的风落之财 (windfall)**" (vol1 L6164) — 收益主要来自土地产出, 不来自傻子馈赠。
- "**尽可能不卖**" (vol1 L6244) — 只有在 "三年后合理估值 150%" **或** "当年 PE 50 倍" (二选其一较低者) 时才触发卖出。默认姿态是 呆坐。
- "**账户里每一毛资本都是自己多年辛苦赚来的**" (vol2 L15452) — 防自欺心态。
- "**下跌的时候, 一个现金流大过净利润, 毛利率高达九成以上的生意, 十五六倍市盈率照样被人当作垃圾一样丢弃**" (vol1 L6701) — 市场先生常态。

## §B. 商业模式 分析 depth

### B.1 核心 商业模式 问题 (不是通用 10 问, 是老唐实际问的)

老唐在每家公司分析里反复追问的 **sharp** 问题 (从 Moutai / 分众 / 洋河 / 海康 案例中归纳):

**关于产品本身**:
1. 产品的 **物理约束** 是什么? (茅台: 酱香 → 空间; 浓香 → 时间。基酒 5 年储存。12987 工艺. vol1 L5650)
2. 产品是 **社交 / 情感 / 收藏 / 功能** 中的哪一类? 各类的需求稳定性不同。茅台是 "社交产品, 强者愈强" (vol1 L4393).
3. **消费者画像**: 真实买单人群是谁? 换一群人还会买吗? (Moutai 反腐 测试, vol1 L6756-6766: 是 "官员爱喝" 还是 "有权者才买得到, 其他群体也有爱"? 决定 反腐 冲击是永久毁灭 还是临时换人。)

**关于定价权**:
4. 公司对客户是 **强势** 还是 **弱势**? (老唐分众 "强" 的三维度: 对客户 / 对上游 / 对手. vol1 L16597)
5. 提价史: 过去 10 年提价频次、幅度, 提价后是否丢客户? (分众 "每年提价 15%", vol1 L16602; Moutai 提价 200 元 毛利率只增 0.8 个点 → 毛利已经很高, vol1 L5691-5693)
6. **出厂价 vs. 终端批价** 的差额。如果差额巨大且持续, 要么公司定价错 (留出空间待调), 要么渠道利润不健康 (将吸引竞争或监管)。Moutai 出厂 969 vs. 一批 2000+ = 老唐 2019-2020 年"三年后必提价" 判断的依据 (vol1 L6820-6889).

**关于竞争 / 商业模式本身**:
7. 有没有 **大、准、强、省、专** 的网络 / 品牌 / 工艺效应? (老唐对 分众 的 5 字 moat 框架, 见 §C.1)
8. 扩大规模能否 **不摊薄毛利**? 边际产能增加的单位成本是上升还是下降?
9. **替代品** 的距离: 不只看当前对手, 更要看 "消费者打发同样需求的替代方案"。(e.g., 电梯广告 vs. 手机广告 vs. 楼下电视 — 老唐倾向于 "互补" 而非 "替代", vol1 L16706).

### B.2 好生意 vs. 坏生意 的具体清单

老唐多次提到的 **好生意** 共同特征 (vol2 L2099-2168 等; 信立泰 / 双汇 / 涪陵榨菜 / 粤高速 等案例):

```
好生意:
  资产结构干净                  (非 "资产 70% 是应收 + 商誉 + 在建工程")
  现金流状态良好                (经营现金流 ≥ 净利润)
  应收账款极少甚至没有          (粤高速 应收 ≈ 0)
  存货管理符合经营需求
  较大数额的预收或应付等经营性占款 (反映议价权)
  销售费用占比低且稳定下降       (反映品牌 pull 而非 push)
  毛利率稳定高位                (= 定价权 + 产品差异化)
  ROE 持续 > 15%                (且不依赖高杠杆)
  分红 "可以分光全部利润"       (不需要持续再投入)

坏生意 (典型信号):
  ROE < 8% 或 波动剧烈           (老唐点杀 门槛: 连续 3 年 ROE > 8%. vol1 L2357)
  毛利率持续下滑
  经营现金流净额 < 净利润        (卖了货收不到钱)
  应收账款 / 营收 > 同业均值
  商誉占净资产比例高
  在建工程长年不转固            (资本支出 "黑洞")
  销售费用增长 > 营收增长        (增长靠推力, 不是拉力)
  "坐庄式" 股权激励 / 定增     (老窖 "无耻定增" vol1 L135)
  管理层发言与业绩数据脱节
```

**点杀标准** (老唐给读者筛股的 "最低门槛", vol1 L2357, vol2 L15531-15549):
```
① IPO > 3 年 (首选 > 5 年, 以获得连续审计数据)
② 最近 3 年 ROE 均 > 8%, 连续 3 年 ROE > 15% 优先
③ 当前 PE < 40 (超过 40 倍意味着需要 3 年利润翻 3-5 倍, 不合理假设)
④ 排除行业: 医药、汽车、地产 (老唐"看不懂")
```

## §C. 护城河 分析

### C.1 老唐 对 护城河 的 分类

老唐 **没有** 构建过巴菲特 / Morningstar 那种 5 分类理论。他分析护城河的方式是 **case-by-case 拆解**。但从案例中可归纳几个反复出现的类型:

| 类型 | 案例 | 关键特征 |
|---|---|---|
| **有效规模** (网络 + 地域自然垄断) | 分众 (梯媒)、白云机场、雅砻江水电、粤高速 | 单终端无价值, 组网后跳跃式增值; 政府不可能批第二家。vol2 L549, L1026 |
| **品牌 + 定价权** | 茅台、五粮液 | 消费者主动买单, 提价不丢客户; 品牌代表 "信息传达 + 时间节约", 对时间宝贵的人价值高 (vol1 L5755) |
| **不可复制工艺 + 时间** | 泸州老窖 (老窖池)、茅台 (茅台镇水源 + 12987) | "再多金钱也无法复制" (vol2 L8003). 时间是护城河的一部分 |
| **管理 + 效率** | 海澜之家、双汇 | 本身商业模式可能不是超 A, 但管理远优于同行, 榨出 "定价以外" 的利润 (vol2 L21904) |
| **社交 + 网络效应** | 腾讯 | "**看不见边际**" (vol1 L204). 社交关系的迁移成本极高 |

### C.2 大、准、强、省、专 — 老唐的 moat 验证框架 (分众案例提炼)

这是老唐最具操作性的 moat 框架 (vol1 L16495, 16521 梳理分众传媒(三)):

```
大   ①规模同行业最大 (给出市占率数字, 如 95% / 70% / 55%)
     ②市场空间还大 (有 headroom, 非天花板)

准   ①受众画像准 (针对性, 能匹配广告主需求)
     ②广告投放准 (转化率 / 触达率 能量化)

强   ①强迫性 / 场景垄断 (消费者无法选择替代渠道)
     ②强势 (对客户 / 对上游 / 对手 都强势 — 三维验证)

省   成本结构碾压对手 (分众不生产内容, 省掉版权 + 明星费用)
     租赁成本 << 内容制作成本

专   专业 / 聚焦 (不是样样都做, 不贪多 — 江南春剥离副业)
     行业最有实战经验的人才 / 模式积累
```

### C.3 如何 **验证** 护城河 (而非 声称)

老唐最常用的 "reality check" 手段:

1. **ROE / 毛利 指标 作为 "路标"** (vol1 L16490-16494): "ROE 指标是个路标, 它的作用是指引我们去寻找那些不在报表里的资产。分众 ROE 维持在 70%, 一定有重要的资产不在资产负债表里。"

2. **提价能力测试**: 历史上每次提价，客户流失吗? 销量涨吗? (Moutai: 8 次提价, 每次销量都涨; 分众: 年提 15%, 客户数年增.)

3. **对手 测试**: 同行业 top 对手盈利能力如何? 若行业 second-best 都在亏损, 说明 leader 的 moat 远超账面。(分众 L16641: 新潮传媒 "规模远大于任何物业或业委会, 净利润依然为负" → 证明分众的 "组网价值" 真实)

4. **反向验证 (消费者 必须)**: 切换成本具体是什么? 可不可以举一个 "我是客户, 我为什么不换" 的具体场景?

5. **"组网价值" 增值 是否能被 leader 独占?** (vol1 L16631): "这份增值由组网者建立, 当然应该由组网者享受"。关键问题: 如果组网价值跳跃式, 但进入壁垒低, 新对手也能快速成网, leader 仍无 moat.

### C.4 ROE 杜邦 分解

老唐对 ROE 杜邦 的态度 **务实而保守**:
- 他不会细到逐年 杜邦 三项画折线 — 他看的是 **"连续 3-5 年 ROE 是否稳定在 15%+"**.
- 他关心的杜邦细节: **净利率 (反映定价权)、总资产周转率 (反映运营效率)、财务杠杆 (反映风险)**. 若 ROE 高但杠杆也高 (银行、保险), 需要对三前提做特别审查 — "贷/存比" "拨备覆盖率" 等。
- 他反复强调: **高 ROE 的秘密通常不在杜邦三项里, 而在 "未被计入账本的资产"** (e.g. 茅台镇水源、分众的终端网络、腾讯的社交链). 杜邦是起点, 不是终点。
- 他会做一个 **逆算 ROE**: "如果茅台把 1000 亿现金分掉, 净资产收益率会高到惊人, PB 也会大到惊人" (vol1 L5591). 这暗示财务结构里"沉淀冗余净资产"会稀释 ROE, 并非企业真实盈利能力。

## §D. 财报 排雷 细节

### D.1 老唐 最 强调 的 红旗 signals (按严重性排序)

来自 vol1 獐子岛 / 雅戈尔 / 乐视网 / 海润光伏 / 蚂蚁 等 "避雷要诀" 系列案例:

**致命红旗 (看到立刻删除观察名单)**:
1. **生物资产**: "农、林、渔、牧 = 造假高危区" (vol2 L15287). 獐子岛 扇贝"跑了" 两次.
2. **商誉/净资产 > 20%**. 后续任何年度都可能一次性计提, 直接干掉利润。
3. **关联交易 + 股权投资 + 炒股收益 占净利润 > 30%** (雅戈尔案例). 投资者实际上买了一个 "炒股账户", 而非实业。
4. **现金流量表负的 / 大幅小于净利润**, 且持续多年 (非短期行业周期). 表示利润是纸面的, 收不回现金.
5. **管理层道德红旗** (vol2 L21452): "管理团队的历史决策所依据的道德观和价值观, 绝对是我们排除企业的重要依据之一。这或许可能让我们错过牛股, 但同样会保护我们避免掉进一些可怕的陷阱。" (老唐 "不喜欢马云" → 避开蚂蚁暂缓上市陷阱. vol2 L21446)

**重要红旗 (减分项)**:
6. 预付账款 / 其他应收款 超常增长 (vol2 L871, 联美控股 10 亿其他应收款 = 控股股东借款). 这些科目是 "大股东占款 / 关联方输血" 的经典伪装点.
7. 在建工程 长年不转固定资产. 扣折旧/摊销就是利润, 不转固就是 "把成本挂起来不确认" — 伪装利润.
8. **销售费用 大幅增长 > 营收增长** (增长靠推力).
9. 应收账款 / 营收 比值 持续上升 (先卖货, 收款越来越慢 → 渠道压货).
10. **高送转 + 减持** 组合 (海润光伏 "高送转陷阱" vol2 L15418).

### D.2 特定 科目 分析 清单 (老唐实战查点)

| 科目 | 老唐 问的 具体 问题 |
|---|---|
| **货币资金** | 和 "有息负债" 比, 如果账上大把现金却还有大量有息负债 → 现金疑似虚假 / 受限 / 被挪用 |
| **应收账款** | 增幅 > 营收增幅? 账龄结构? 前 5 大客户集中度? 坏账计提比例 是否激进/保守? |
| **应收票据** | 银票 vs. 商票占比? 商票违约风险. Moutai 案例: 应收票据 飙升往往对应 销售政策放松 (vol1 L1746) |
| **预付账款** | 给谁预付的? 是否常年挂账? (关联方通道?) |
| **其他应收款** | **查关联方**. 数额大 + 挂账长 = 重大疑点 (vol2 L871 联美控股 案例) |
| **存货** | 跌价准备 计提充分? 存货周转率 vs. 同业? 生物资产 / 高价原料 → 造假区 |
| **在建工程** | 总额 vs. 固定资产 比例. 长期不转固 = 挂账操纵折旧 |
| **商誉** | **商誉/净资产 > 20% 视为雷区**. 每年减值测试披露看假设 (revenue growth / WACC) 是否激进 |
| **预收账款 / 合同负债** | **正向指标** (客户提前付钱 = 议价权). Moutai 把它作为调节报表工具, 需要 "还原" (vol1 L1573) |
| **其他流动资产** | 主要是 银行理财, 视为 类现金 |
| **可供出售金融资产 / 交易性金融资产** | "浮亏"或"浮盈"要核实, 不能信账面. (雅戈尔 中信股份浮亏 案例 vol2 L15330) |

### D.3 现金流 真伪 验证 方法 (老唐最犀利的武器)

出处: vol2 L21548-21560, 以 Moutai 为例的推导:

```
销售收现 = 营业收入 + 增值税 (17%) 
         - 应收账款增加额 
         - 应收票据增加额 
         - 贴现应收票据付出的财务费用 
         + 预收账款增加额
```

**应用方法**: 对目标公司, 用这个公式算出 "应该收到多少现金", 然后和现金流量表里 **"销售商品、提供服务收到的现金"** 逐年对比。
- 若 实际 > 公式值 → 现金流真实, 可能还有额外回款;
- 若 实际 显著 < 公式值 → 有 "纸面营收" 嫌疑, 进一步查应收 + 票据结构.

**还原真实销售** (排除预收款操纵): vol1 L1573-1574:
```
剔除预收款后的真实销售 = 营业收入 + (期末预收款 − 期初预收款) / 1.17
```
Moutai 案例: 2016 年报表营收 389 亿, 预收款净增 92.8 亿, 还原后真实销售 = 389 + 92.8/1.17 = 468 亿 (vol1 L4362).

**经营现金流 vs. 净利润**: 老唐的经验值
- "经营现金流 ≥ 净利润" 是及格线;
- **"经营现金流 >> 净利润"** (如 2 倍, 说明利润被折旧/摊销低估) = 优质企业信号 (vol2 L491, L8957);
- **"经营现金流 < 50% 净利润"** 持续 2 年以上 = 红旗.

## §E. 管理层 评估

### E.1 看年报的哪些部分识别管理层品质

1. **"致股东信"/ 管理层讨论与分析** — 语气是否自大 / 自省? 是否直面坏消息? (Moutai 袁仁国 2014 "3% 增长" 预告, 实为 "被迫", vol1 L749)
2. **业绩预告 vs. 实际** — 历年偏差方向和幅度. Moutai 2009-2013 五年预算报告 vs. 实际完成 (vol1 L981) 是老唐经常用的信心 vs. 兑现 对照表.
3. **信息披露质量** — 产量 / 销量 / 库存 / 产品结构细节是否披露 (vol1 L6438-6443: Moutai 李保芳 任内 "信息披露质量翻天覆地").
4. **分红率 + 股权激励** — 见 §E.3.
5. **关联交易** — 规模, 定价机制, 公众股东否决权. (Moutai 章程 第 79 条, vol1 L5486)
6. **"膀胱效应"** — 本应分给股东的利润被管理层胡乱投资 (vol1 L1317).

### E.2 言行一致 的 verification 方法

这是老唐最强调的。核心操作:
1. **找历史访谈 / 新闻稿 / 业绩发布会** 里管理层的 "承诺" 或 "判断" — 扩产进度、提价计划、新业务目标、行业展望。
2. **建一张 Excel** (老唐 自己就这么做的): 每年管理层承诺 vs. 年底实际达成. 差异大的年份标红. 系统性乐观 / 系统性保守, 都是 signal.
3. **历史多年 的轨迹 决定可信度** — 不看单次, 看连续 5-10 年模式。李保芳 "允许系列酒政策性亏损三年" → 事后 "10 倍营收" 兑现 (vol1 L6476-6491), 所以老唐给李董打高分.
4. **承诺 vs. 激励** 一致性: 股权激励方案的业绩考核线, 是否 贴合 管理层公开目标? 太低 (送钱) or 太高 (无法兑现) 都是红旗.

### E.3 历史 预测 vs. 兑现 的 tracking 操作 (老唐最具体的工具)

Moutai 管理层评估 case study (vol1 L975-988):
```
表 2: 茅台 管理层 预算报告 vs. 实际 (2009-2013)
  2011 信心满满 → 现金流指标显示 未来强增长 → 2011 实际超预算
  2012 信心膨胀 → 动用 预收款 强行营收 +44% → 为 2013 埋雷
  2013 信心碰壁 → 反腐+限额 → 前高后低, 曲线从"持续成长"扭曲为"日落西山"
```
**教训**: 管理层信心周期与业绩周期同步, 顶峰时乐观, 拐点时不认. 要用**多年窗口**看, 不看单年。

**分红率 评判** (vol1 L1300-1317, 回复 赵丹阳 关于分红的讨论):
- 分红率 **不是** 越高越好, 要结合 ROE 看.
- "挣的多 + 分的少" (ROE 高 + 分红率低) = 管理层把股东的钱再投入高回报项目, **好**.
- "挣的少 + 分的多" (ROE 低 + 分红率高) = 没有好项目, 把钱发回股东, **也好, 至少诚实**.
- "挣的多 + 分的少 + 再投入回报低" = **烂管理层** (膀胱效应).
- 判断再投入回报的方法: 历年 留存收益 ÷ 净利润增长 比率.

## §F. 老唐估值法

### F.1 公式 derivation

**两句话**: "三年后 15~25 倍 PE 卖出能赚一倍的价格, 就可以买入。高杠杆企业打七折。" (vol2 L2985-2986)

**完整步骤** (vol1 L6921, Moutai 2020.4.27 演示):

```
Step 1: 预估 "3 年后 的 归母 净利润"
        — 分业务 板块 拆解 (Moutai: 茅台酒 + 系列酒 + 财务公司)
        — 对每块做 量 × 价 × 净利率
        — 给一个 上下限 (保守 vs. 乐观) → 取中值 ± 10%

Step 2: 给 合理 PE 倍数
        合理 PE = 1 / 无风险收益率
        无风险收益率 3%~4% → 合理 PE 25-30 倍 (vol1 L10317, L5810)
        对于 "三大前提 都 确认 + 自由现金流 > 利润" 的 顶级企业, 取上限 30 倍
        一般企业 取 25 倍
        **重点: 取值与企业利润增速无关, 只与无风险收益率相关** (vol1 L5810)

Step 3: 3 年后 合理估值 = 3 年后净利润 × 合理 PE
        以 Moutai 为例 (2020.4.27 计算):
        三项合计 628-788 亿 净利 → 700 亿 ± 10%
        合理 PE 27.5 (取 25-30 中值)
        → 合理估值 = 700 × 27.5 × ± 10% ≈ 19000 ± 10% 亿

Step 4: 买点 = 3 年后合理估值 × 50%
        = 19000 / 2 = 9500 亿 (Moutai 2020.4.27 老唐买点)
        高杠杆企业 打 7 折, 即 × 35%

Step 5: 卖点 = min(3 年后合理估值 × 150%, 当年净利 × 50 PE)
        ① 3 年后合理估值 × 110%(安全边际) × 150% = 31350 亿
        ② 2020 年估计净利 460 亿 × 50 PE = 23000 亿
        取较低: 23000 亿 = 卖点
        分批卖: 达 23000 亿卖 1/3; 再涨 10% 卖 1/3; 再涨 10% 清仓 (vol1 L7045)

Step 6: 持仓区间 = [买点, 卖点]
        此区间内 持股不动, 收工睡觉.
```

### F.2 参数 选择 细节

**无风险收益率**: 老唐取 5 年期国债 (vol2 L2983) 或 当前市场 10 年期国债收益率。2013-2020 多年间他用 3.5%-4%。2020 后有所下调 (低利率环境)。
- 若 利率 上升到 5%, 合理 PE 应降到 20; 利率下降到 2%, 合理 PE 可到 50.
- 他反复警告: **"老唐估值法, 仅适用于符合三大前提的企业"** (vol1 L6184). 对不符合的垃圾股, "举着火把穿越炸药库", 估值无意义。

**50% 打折 (买点 = 合理估值 × 50%)** 的逻辑:
- 三年赚 100% → 年化 26%. 若无法承受估值误差, 选此.
- 高杠杆企业打 7 折 (即 35%) — 因为三前提 的"持续是否依赖资本投入"更脆弱.

**150% 溢价 (卖点 = 合理估值 × 150%)** 的逻辑:
- vol1 L6314-6323: 假设企业 25% 增长 + 4% 无风险收益率. 50 PE 卖出后, 3 年无风险 = 本金 × 1.125. 3 年后企业净利 × 1.95 × 25 PE = 新价格。用新价重买, 能多买 ~15% 股数.
- 若回落早于 3 年或增长低于 25%, 占便宜更多.

## §G. Moutai specific insights ★ (seed for Part 1 deepening)

### G.1 老唐 对 Moutai moat 的 unique framing

**超越 "品牌" 的 4 维 moat** — 老唐把 Moutai 护城河拆得远比券商研报细:

1. **空间的稀缺性 (地理 moat)** — 茅台镇 7.5km² 核心产区 + 中华村扩产区. 酱香不能异地复制 (vol1 L5650). 基酒年产天花板 5.6-7 万吨 (vol1 L5570: "保芳董 公开说过 5.6 万吨后不再考虑扩产"). **这是最硬的 moat — 无法被资本 / 技术复制**.

2. **时间的稀缺性 (工艺 moat)** — 12987 (1 年周期 + 9 次蒸煮 + 8 次摊凉 + 7 次取酒). 基酒储存 5 年. 每年基酒挥发约 2% (vol1 L5712, 季克良 披露). 这决定了 **产销 与 4-5 年前 基酒产量 强相关** — "四年前基酒产量 × 75%-85% = 当年可售商品酒数量" (vol1 L5745).

3. **社交 / 收藏 需求** (非 "喝" 需求) — 老唐明确拒绝 "收藏不是真实需求" 的说法 (vol1 L5644): "存新酒喝老酒, 将会是长期存在的需求". 这层需求 **与 GDP 强相关, 但与健康趋势弱相关**.

4. **消费者扩散 moat** — "强者愈强" (vol1 L4393): 喝的人越多, 越容易被选择 (社交证明). 反腐 不能毁掉茅台, 因为"官员"不是群体属性, 只是"获取渠道"属性 (vol1 L6762).

### G.2 跟踪的 关键 数据 (老唐 Moutai 季度/年度 excel)

从 vol1 整理出 老唐 tracks 的 Moutai 核心数据:

```

  □ 储存空间 (旧酒库 + 新酒库)

销售 / 需求侧:
  □ 当年茅台酒商品酒销量 (吨)
  □ 当年系列酒销量 (吨) + 营收
  □ 出厂均价 (综合均价, 不只是普茅 969)
  □ 直销比例 (直销价 ≈ 1499, 是普茅 出厂价的 1.55×, 占比提升 = 均价提升)

  □ 经销商数量变化 (整顿 → 直销转化)

财务 操纵 / 真实度:
  □ 预收账款 / 合同负债 (季末余额, vs. 营收的比率)
  □ 应收票据 余额 + 结构 (银票 vs. 商票)
  □ 销售收现 / 营业收入 比值 (应 > 1 + 增值税率)
  □ 销售费用 绝对值 和 占营收比 (茅台正常 4-5%, 异常时需深查)
  □ 分红率 和 绝对分红金额 (期望 每股 17.5 元+)

产品结构:
  □ 普茅 vs. 非普茅 (小批量 / 陈年 / 纪念) 占比
  □ 系列酒: 王子 / 迎宾 / 汉酱 / 仁酒 各自销量 + 营收 + 毛利
```

## §I. 投资哲学 基础 (格巴芒 + 老唐 的 底层 mental models)

本节 补录 唐朝 体系 里 "先于 分析" 的 世界观: 企业 = 什么, 为什么 只看 自由现金流, 什么 是 "看得懂", 什么 是 "长坡厚雪". 这些 看似 抽象, 但 每一次 具体 估值 动作 都 在 默认 它们 已经 通过.

### I.1 "下蛋的母鸡" 视角 — 企业 = 持续产生现金流的资产

出处: 投资哲学卷 L8562-8629 (老唐 解释 自由现金流折现 的 口语化 演绎, 术语 "下蛋的母鸡" 就是 他 本人 的 用语).

核心思想: 把 一家 企业 当做 一只 老母鸡 看. 投资 100 元 买下 这只 母鸡, 未来 的 总回报 = (每天 下的 蛋售价 − 每天 吃的 饲料) 合计 + 最后 老了 炖汤 卖掉 的 残值. 中间 每一个 蛋带来 的"蛋钱 − 饲料钱"差额 就是 老唐 定义 的 **自由现金流**: 不是 "经营现金流 − 全部 CapEx", 而是 "经营所得 − 维持 现有 盈利能力 所需 的 资本支出" (投资实战卷 L6856).

**操作 含义**:
- 任何 公司, 先问 "它是 不是 一只 会 下蛋 的 母鸡? 还是 一只 需要 我 不断 喂 更多 饲料 才能 继续 下蛋 的 母鸡?" — 这 直接 对应 三大前提 的 第三项.
- "饲料钱" 必须 足额 扣除 — 折旧摊销 不足 以 覆盖 真实 维持性 CapEx 的 行业 (e.g. 重资产 制造、航空、钢铁), 报表 净利润 ≠ 可分红 的 真实 收益.
- 母鸡 视角 阻断 了 "市值 / 股价 / 市梦率" 这些 与 现金流 无关 的 幻象. 唐朝 反复 说 "**All cash is equal** — 没有 哪个 行业 赚到 的 钱比 其他 行业 低贱 或 高贵, 它们 只有 数量 差异, 没有 质地 差异" (哲学卷 L9230).

母鸡 视角 + "三阶段 现金流 折现 模型" → 投资哲学卷 L8621-8629 给出 的 三步演绎:
```
1. 估算 青年母鸡 产蛋高潮期 每年 自由现金流, 折现加总
2. 估算 中年母鸡 产蛋低潮期 每年 自由现金流, 折现加总
3. 估算 老年母鸡 丧失产蛋能力 后 的 残值, 折现
  → 三步 相加 = 企业 内在价值
```
但 唐朝 立即 在 原文 指出: **公式 不重要, 思维 重要**. "动用 公式 去 计算 未来 自由 现金流 折现值 是 一条 错误 的 道路" (哲学卷 L8640). 真正 有用 的 是 这套 思维 模式 指引 投资者 去 找 "每年 可以 产生 可观 的 自由现金流、不依赖 大量 资本 投入 即可 维持 现有 盈利能力 甚至 还 可以 保持 增长" 的 企业 — 这是 为什么 后面 的 老唐估值法 (§F) 敢于 用 极简 的 "净利润 ≈ 自由现金流 × 25 倍" 近似.

### I.2 "长坡厚雪" 与 "慢变"

"长坡厚雪 慢变" 不是 唐朝 原创 (源自 巴菲特 对 滚雪球 的 比喻), 但 唐朝 反复 把 它 作为 选股 第一层 筛子:
- **坡要长**: 行业 未来 至少 10 年 仍然 存在, 需求 不会 被 技术 替代 或 消费 偏好 淘汰 (白酒、消费 品牌、长周期 公用事业 合格; 日新月异 的 消费 电子、政策 敏感 的 新能源 补贴 红利 不合格).
- **雪要厚**: 毛利率 + ROE 长期 高位 — 不是 "风口" 给 的, 是 商业模式 结构 给的 (茅台 品牌价格带、分众 组网效应、信立泰 仿制药 高毛利 仿制 壁垒).
- **慢变**: 行业 的 技术 / 监管 / 消费者 偏好 演变 节奏 足够 慢, 让 投资者 有 几年 的 思考 窗口. 这点 直接 引出 唐朝 "排除 医药 / 汽车 / 科技" 的 保守 倾向 (投资实战卷 L2357 点杀 标准 第④条).

**操作 含义**: §1 商业模式 / §2 成长空间 收尾 的时候, 必须 自 问 一句 "这家公司 10 年后 还 存在 吗? 在 做 的 事情 本质 上 变 了 吗?" 答不出 "存在 且 大概率 仍做 同样 的事" → 不适合 用 25PE 估值法, 必须 更深 折扣 或 弃权.

### I.3 "看得懂" 的 标准 — 能力圈 四问

出处: 投资实战卷 L5402-5416 (原文 题 目 《看得懂 与 看不懂》, 2017 年发表, 后 被 反复 引用). 唐朝 把 "看得懂 一家公司" 定义 为 能 完整 回答 以下 **四个 问题** (不是 三个, 是 四个 — 注意 与 俗传 的 "三问" 区分):

```
① 这家 公司 靠 销售 什么 商品 / 服务 获取 利润?
② 它的 客户 为何 从 它 这里 采购, 而 不 选 其他 机构 的 商品 / 服务?
③ 资本 的 天性 是 逐利. 为什么 别 的 资本 没 提供 更 高 性价比, 抢走 它 的 市场 份额
  或 逼 它 降低 利润?
   (更高性价比 = 同质量+更低价格, 或 同价格+更高质量)
④ 假设 同行 挟 巨资 / 产业巨头 挟 巨资 参与 竞争, 该公司 能否 保住 乃至 继续
   扩张 市场份额?
```

四问 通不过 = 看不懂 = 能力圈外 = 不 下注 (投资哲学卷 L7217 "跨出 能力圈 下注, 是 注定 要 被 别人 收割 的 — 早晚 而已"). **操作 上**: §1-§3 section worker 提交 定性 分析 前 应 能 口述 这 四问 的 答案; 任一 回答 含糊 就不 允许 推进 §4-§6.

注意 与 "三大前提" (§A.2) 的 区别: 三大前提 是 估值 门槛, 看的是 报表 真伪 + 持续性 + 资本强度. 能力圈四问 是 理解 门槛, 看的是 投资者 本人 对 商业 机理 的 把握程度. 两者 同时 必过, 缺一不可. 唐朝 的 原话 (投资实战卷 L5401) 是 "知 与 不知 的 分水岭".

### I.4 "一流 生意、三流 管理层" 与 好生意 优先级

出处: 散落 在 投资哲学卷 关于 消费品 / 公用事业 的 讨论. 核心 主张: 生意 模式 的 质量 优先级 > 管理层 质量. 一流 生意 + 三流 管理层 通常 优于 三流 生意 + 一流 管理层, 因为:
- 一流 生意 的 经济商誉 会 让 即使 平庸 的 管理层 也 能 挣到 钱 (粤高速、长江电力 的 案例);
- 三流 生意 + 一流 管理层 = 管理层 被 迫 不断 重组 / 转型 / 跨界, 成功 案例 稀少 且 不可 复制.

**操作 含义**: §1 (好生意) 一旦 判定 "否", 不要 指望 §4 (管理层) 的 亮点 能 救回; 反之, §1 判定 "是" 时, §4 的 平庸 可 接受, 只要 不存在 §4 红旗 (道德 / 大股东占款 / 系统性 画大饼).

### I.5 心态 层: 安全边际 = "我 错了 也 不会 输", 不是 "便宜"

投资哲学卷 L7071-7073 给出 了 老唐 对 安全边际 的 关键 重新定义: 安全边际 **不是** "价格 低于 内在价值" 那么 简单, 而是 "**即使 我 预计 错了, 损失 也不大**". 两者 的 区别:
- 俗版安全边际: 我 估 内在价值 100 元, 现在 价格 50 元, 打 五折 买入 = 便宜.
- 唐朝版安全边际: 我 估 内在价值 100 元, **但 我 可能 估错**; 即使 真实 价值 只有 70 元, 我 50 元 买入 仍然 不会 永久 亏损. 50% 折扣 的 意义 是 给 自己 **估算 错误** 留空间, 不是 给 市场 错判 留空间.

**操作 含义**:
- 买点 = 3 年后 合理估值 × 50% 这个 折扣, 50% 中 的 大部分 不是 为 "占便宜", 而是 为 "我 可能 判错". 如果 判错 概率 高 (认知 不深、行业 变化 快、管理层 难评), **应 提高 折扣** 而不是 降低 门槛买入.
- 这 也 解释 为什么 唐朝 的 "收工睡觉" 区间 里, 持股 下跌 不必 加仓: 下跌 到 买点 以下 的 时候, 应 先 反思 "是不是 我 估错 了", 而 不是 默认 "市场先生 在 发癫".
- 默认 规则 (哲学卷 L6244 "尽可能 不卖"): 如果 自己 对 企业 的 理解 没变 而 股价 下跌, 不加仓 = 承认 自己 判断 可能 错了; 不卖出 = 但 也不 因为 一次 下跌 就 放弃 判断. 加仓 的 前提 是 "我 又 重新 做 了 一遍 功课, 确信 企业 本身 没 变".

### I.6 心态 层: 市场先生 与 "呆坐 / 收工睡觉"

投资哲学卷 L3131-3167 大段 引用 了 格雷厄姆 的 "市场先生" 寓言, 唐朝 反复 用 它. 操作 层面 的 关键 推论:
- 市场 报价 每天 变化, 但 企业 内在价值 几年 才 变 一次 — 所以 大部分 时间 正确 的 动作 是 "**呆坐 不动**" (哲学卷 L4762 "我 喜欢 市场先生, 喜欢 他 撒娇, 喜欢 他 发癫").
- "尽可能 不卖" 默认 姿态 (哲学卷 L6244): 只在 **三年后 合理估值 × 150%** 或 **当年 PE 50 倍** 触发 卖出, 二者 取 低者. 其他 时间 一律 持有.
- 股价 上涨 不是 卖出 理由, 估值 超区间 才是; 股价 下跌 不是 买入 理由, 估值 进入 买点 才是.
- "牛市 只是 投资者 的 风落之财 (windfall)" (哲学卷 L6164) — 长期 复利 来自 企业 内生 现金流, 不 来自 "傻子 馈赠".

### I.7 避免 永久性 亏损 > 追求 高 收益率

多处 反复强调. 排序 上: **永不 永久性 亏损 > 年化 收益 > 短期 博取 高点**. 这 决定 了 唐朝 方法论 的 整体 保守 倾向:
- 宁可 错过, 不愿 错买. 能力圈 外 一律 不碰, 即使 热门.
- 估值 前置条件 (三大前提) 任一 "假" → 降级, 不估值, 不 买入.
- 买点 × 0.5 的 折扣 是 永久 原则, 不 因为 "这次 特殊" 打折.

## §J. 财报 阅读 方法 (三大表 的 唐朝 读法)

唐朝 《手把手 教你 读 财报》两卷 的 核心 骨架. 此处 做 操作 化 提炼, 完整 推导 见 原书. 每 张 表 的 读法 对应 的 是 "三大前提" 里 的 不同 角度:
- **资产 负债表 = 看 家底** → 对应 "利润为真" 的 静态 侧面 + "持续性" 的 结构 侧面
- **利润表 = 看 本期 赚多少** → 对应 "利润为真" 的 流量 侧面
- **现金流量表 = 核验 真假** → 对应 "利润为真" 的 交叉 验证 + "资本强度" 的 实操 度量

### J.1 资产 负债表: 看 家底

老唐 的 核心 动作 是 **"简化 资产 负债表"** — 把 报表 上 几十 个 科目 压缩 到 5 大类 (投资实战卷 L10709 信立泰 案例 是 经典 演示):

```
类现金          = 货币资金 + 应收票据(银票部分) + 其他流动资产(理财) + 交易性金融资产
                + 部分 可供出售 金融资产
经营资产        = 应收账款 + 存货 + 预付账款(扣关联方) + 合同资产
生产资产        = 固定资产 + 在建工程 + 无形资产(土地/软件 等 真实生产要素) + 使用权资产
投资资产        = 长期股权投资 + 投资性房地产 + 其他非流动金融资产
商誉 / 无形(软)  = 商誉 + 无形资产里的 品牌 / 客户关系 / 技术授权 等 会计 推算物
```

**判断 准绳 (一看 就 知)**:
- **类现金 占 总资产 > 50%** 的 消费品 公司, 且 总市值 ≈ 类现金 + 欠条盈余 → "市场 在 白送 生意" (信立泰 低估 时 的 经典 形态, 投资实战卷 L11256).
- **商誉 / 净资产 > 20%** → 雷区 (随时 可能 计提 一次性 减值, 直接 毁 当年 利润).
- **经营资产 (应收 + 存货) 占 总资产 的 比重 长期 上升** → 要么 是 放宽 信用 抢 销量, 要么 是 压货. 交叉 核验 销售收现率.
- **在建工程 长年 不转 固定资产** → 挂账 操纵 折旧 (不转 固 就 不 计提 折旧, 利润 被 虚抬).
- **生产资产 占比 高** (超过 40-50%) → 重资产 生意, "母鸡 需要 不断 喂 饲料", 维持 性 CapEx 压迫 自由现金流, 三大前提 第三项 易 不过.

**附注 层的 家底 清单 (逐项 读, 不要 跳)**:

| 附注 项 | 读什么 | 唐朝 红旗 |
|---|---|---|
| 货币资金 | 受限 资金 比例 (定期 / 保证金 / 质押) | 大额现金 + 大额 有息负债 并存 → 货币资金 可能 被 挪用 / 虚假 (康美 类) |
| 应收账款 | 前 5 大 客户集中度 + 账龄结构 + 坏账计提政策 | 应收 增幅 明显 > 营收 增幅; 账龄 结构 恶化 (一年以上 占比 上升); 坏账政策 显著 宽松于 同业 |
| 应收票据 | 银票 vs. 商票 占比 | 商票 占比 突然 上升 → 收款 质量 下降 (银票 有 银行 背书, 商票 只有 购货方 信用) |
| 预付账款 | 给谁 预付、期限 | 长期 挂账 关联方 (可能 输血) |
| 其他 应收款 | 对象 + 金额 | **大股东 / 关联方 占资 的 经典 伪装点**. 金额 > 流动资产 10% 且 对 单一 关联方 挂账 长年 → 强红旗 |
| 存货 | 分项 (原材料 / 在产品 / 库存商品 / 发出商品) + 周转率 + 跌价准备政策 | 跌价准备 计提 明显 弱于 同业; 存货 增速 远 > 营收 增速; 高价 原料 / 生物资产 → 造假 高危区 |
| 固定资产 + 在建工程 | 年增量 + 转固 节奏 | 在建 工程 数 年 不转固; 折旧 政策 明显 比 同业 慢 |
| 商誉 | 减值测试 假设 (营收增长率 / WACC) | 假设 明显 乐观 (营收 增长 > 10% 假设); 历年 未 减值 但 被 收购 资产 实际 营收 下滑 |
| 合同负债 / 预收账款 | 绝对额 + 占 营收 比 | **正向 指标** — 客户 愿意 提前 付款, 反映 议价权. 但 同时 要 警惕 被 用作 调节 报表 工具 (茅台 案例 vol1 L1573) |
| 应付账款 + 应付票据 | 占 上游 应收 比例 | **正向 指标** — 对 上游 的 议价权 |
| 长期股权 投资 / 可供出售 金融资产 | 公允价值 vs. 账面; 炒股 / 股权 投资 占 净利 比重 | 炒股收益 > 30% 净利 → 投资者 实际 买了 一个 "炒股账户" (雅戈尔 警示) |
| 有息负债 (短借 + 长借 + 应付债券) | 绝对额 + 与 经营现金流 对比 | 有息负债 / 经营现金流(近3年) > 3 → 高杠杆, 估值 × 35% 而不是 × 50% |

### J.2 利润表: 看 本期 赚多少

利润 真伪 的 三条 主线:

**主线 ①: 利润 构成 拆解**
- 扣非 净利润 / 净利润 ≥ 0.85 — 否则 利润 被 一次性 非经常 损益 主导, "纸面" 嫌疑.
- 毛利率 长期 稳定 (±3-5 点 内) — 大幅 波动 = 定价权 松动 或 成本 失控.
- 销售费用 / 营收 稳定 或 缓慢 下降 — 持续 上升 = 增长 靠 销售 推力, 不是 品牌 拉力 (老唐 反复 点名 这是 "坏生意" 信号).

**主线 ②: 毛利率 vs. 净利率 跨年 对比**
- 毛利率 涨 但 净利率 不涨 → 费用 膨胀, 销售 / 管理 / 财务费用 里 有 漏洞.
- 毛利率 稳 但 净利率 跳涨 → 查 是否 税率 一次性 优惠、是否 非经常 损益 虚高.

**主线 ③: "真实 销售" 还原** (排除 预收款 调节 + 应收 信用 放松):
```
真实 销售 = 营业收入 + (期末 预收/合同负债 − 期初 预收/合同负债) / (1 + 增值税率)
```
茅台 案例 (投资实战卷 L1573): 2016 年 报表 营收 389 亿, 预收款 净增 92.8 亿, 还原 后 真实 销售 ≈ 468 亿 → 实际 销售 强度 比 报表 更强.

这 公式 反向 也 适用: 如果 公司 连续 几 季度 "还原 后 真实销售 < 报表 营收", 说明 在 透支 预收款 做 报表, 未来 反扎 风险大.

### J.3 现金流量表: 核验 真假 (老唐 最 锋利 的 武器)

核心 公式 — **销售收现 交叉 验证** (投资实战卷 L21548):
```
应有 销售收现 = 营业收入 × (1 + 增值税率)
              − Δ 应收账款
              − Δ 应收票据
              − 贴现 应收票据 付出的 财务费用
              + Δ 预收账款 / 合同负债
```
再 对比 现金流量表 里 "销售商品、提供劳务 收到的 现金" 实际值:
- 实际 ≥ 公式值 → 现金流 真实, 可能 还有 额外 回款;
- 实际 << 公式值 (背离 > 5%) → "纸面 营收" 嫌疑, 深查 应收 + 票据 + 预收 结构.

**经营现金流 vs. 净利润 的 唐朝 经验值** (投资哲学卷 + 投资实战卷 反复):
| 比值 | 含义 |
|---|---|
| 经营现金流 / 净利润 ≥ 1.0 | 及格线. 利润 是 真金白银. |
| 经营现金流 / 净利润 >> 1 (如 ≥ 2) | 优质 企业 信号. 通常 说明 折旧 摊销 额度 保守, 实际 维持性 CapEx < 会计折旧. 茅台 / 分众 典型. |
| 经营现金流 / 净利润 < 0.5 持续 2 年+ | 强红旗. 利润 收不回 现金, 要么 大量 应收, 要么 大量 存货, 要么 造假. |
| 经营现金流 净额 为负 (非 短期行业周期) | 直接 淘汰. 母鸡 不下蛋. |

**自由现金流 的 唐朝 定义** (投资实战卷 L6856, 与 学术 略 不同):
```
自由现金流 = 经营 所得 − 维持 现有 盈利能力 所需 的 资本支出
           ≠ 经营现金流 − 全部 CapEx
```
关键 差异: 维持性 CapEx, 不是 全部 CapEx. 扩张性 CapEx (新建 厂房 扩产) 不减 — 它 是 "未来 更大 只 母鸡" 的 投入. 这 让 唐朝 对 "高 CapEx 但 扩张 性" 的 企业 (如 信立泰 早期、海康) 不 自动 判死; 但对 "高 CapEx 却 只能 维持" 的 企业 (长安汽车, 投资实战卷 L2988-3001), "每年 的 65 亿 CapEx 是 泡沫利润, 需要 去 泡沫化".

### J.4 三大表 勾稽 关系 (核验 清单)

每 一张 年报, 读完 三大表 之 后 必须 做 以下 3 项 勾稽:

1. **净利润 → 未分配利润 变化**: 本期 净利润 − 分红 − 盈余公积 提取 = 未分配利润 本期 增量. 不 平 = 会计 错报 或 漏披露.
2. **货币资金 变化 = 现金流量表 三活动 净额 合计 + 汇率 影响**: 资产 负债表 的 货币资金 期末 − 期初 应 等于 现金流量表 的 "现金及 现金等价物 净增加 额". 不 平 = 存在 未 披露 受限 资金 变动 或 合并报表 口径 异常.
3. **应交税费 与 税前 利润 对应**: 所得税 费用 / 税前利润 应 接近 名义税率 (A股 通常 25%, 高新 15%). 长期 明显 低于 应有 税率 (< 10%) → 查 递延 所得税 项目, 防 "税 的 黑箱".

附注 必读 (超越 正文 的 真实 信息 所在):
- **关联交易 明细** — 交易 规模、定价 机制、公众股东 否决权设置.
- **对外 担保** — 被担保 方 是否 关联方、是否 逾期.
- **或有 事项** — 未决 诉讼、未达 里程碑 的 承诺、对赌 条款.
- **政府 补助** — 占 当年 利润 比重, 是否 可持续; 补贴 依赖 的 企业 不 适用 25PE.
- **商誉 减值 测试 假设** — 假设 是否 激进 (营收 增长率 / WACC 两个 关键 参数).
- **存货 跌价 准备 政策** — 与 同业 比 是否 宽松.
- **应收账款 坏账 政策** — 账龄 区间 的 计提 比率 是否 与 行业 差异 过大.
- **股份 支付 / 股权 激励** — 行权 条件 (业绩 考核 线) 是否 与 管理层 公开 目标 吻合、是否 过低 (送钱).

## §K. 排雷 清单 (20 项 操作 化 检查, 可 直接 作为 reviewer checklist)

§D 已 给出 了 致命 / 重要 红旗 的 分类 陈述. 本节 把 它们 拆成 **可 逐项 打勾 的 20 条 检查项**, 每 条 带 "看哪里 + 触发 条件 + 后续 动作". 专为 §4.5 排雷 section 和 §3 三大前提 设计, 可 直接 嵌 入 子 agent prompt 作为 checklist.

### K.1 资产 类 红旗 (7 项)

1. **商誉 / 净资产 > 20%**
   - 看: 资产负债表 商誉 + 附注 减值 测试 假设
   - 触发: 比值 > 20% 或 当年 商誉 增额 > 净利润
   - 动作: 整体 profile 降 置信度 一档; 减值 测试 假设 激进 (营收增长 > 10%) 再 降 一档.

2. **其他 应收款 异常 (大股东 / 关联方 占款)**
   - 看: 附注 其他 应收款 明细
   - 触发: 金额 > 流动资产 10%, 或 对 单一 关联方 长期 挂账 (≥ 2 年)
   - 动作: 强 红旗. profile 降 一档; 追 回款 计划 披露.

3. **货币资金 与 有息 负债 同时 大额**
   - 看: 资产负债表 + 附注 受限 资金 披露
   - 触发: 货币资金 > 有息负债 × 80% 且 两者 均 > 净资产 30%
   - 动作: 疑似 现金 虚假 / 被 占用; 追问 为何 有 充足 现金 还 借款; 不过 关 则 三大前提 ① = 存疑.

4. **在建工程 长年 不转 固定资产**
   - 看: 近 5 年 在建工程 vs. 固定资产 变化; 附注 工程 进度
   - 触发: 同一 工程 账面 3 年 以上, 或 在建工程 / 固定资产 > 30% 长期 不降
   - 动作: 挂账 操纵 折旧 嫌疑; 还原 真实 折旧 (按 合理 完工 节奏 假设) 重 算 扣非 利润.

5. **应收账款 增幅 明显 > 营收 增幅**
   - 看: 近 3 年 应收账款 增长率 vs. 营收 增长率; 附注 前 5 大 客户
   - 触发: 应收 增幅 > 营收 增幅 的 1.5 倍, 持续 2 年以上
   - 动作: 压货 / 放宽 信用 嫌疑; 查 经销商 库存 + 终端 批价 交叉 验证.

6. **存货 跌价 / 生物资产**
   - 看: 附注 存货 分项 + 跌价 准备政策
   - 触发: 农林 渔牧 主业; 或 存货 年增 > 30% 且 跌价 准备 比例 远 低于 同业
   - 动作: 造假 高危区 (獐子岛式), 默认 直接 降 置信度 两档 或 放弃 估值.

7. **可供 出售 / 交易性 金融资产 浮亏**
   - 看: 附注 公允价值 变动 明细
   - 触发: 炒股 / 股权 投资 收益 占 净利 > 30%; 或 浮亏 占 净资产 > 10% 未 披露
   - 动作: 投资者 实际 买了 一个 "炒股账户" (雅戈尔 案例); 估值 对象 错位, 降 置信度.

### K.2 利润 / 现金流 类 红旗 (5 项)

8. **经营现金流 净额 < 50% 净利润 持续 2 年**
   - 看: 近 3 年 现金流量表 与 利润表 对比
   - 触发: 比值 < 0.5, 连续 ≥ 2 年
   - 动作: 三大前提 ① = 假/存疑; 立即 降级 为 定性 研究, 不估值.

9. **销售 收现 交叉验证 背离 > 5%**
   - 看: 按 §J.3 公式 计算 应有 销售收现 vs. 实际
   - 触发: 实际 比 公式值 低 > 5% 连续 2 年
   - 动作: 营收 真实性 存疑; 深查 应收 + 票据 + 预收 结构.

10. **毛利率 持续 下滑 且 无 合理 解释**
    - 看: 近 5 年 毛利率 趋势
    - 触发: 累计 下降 > 5 点 且 管理层 无 具体 说明 (价格战 / 原料 / 产品结构)
    - 动作: 定价权 松动 信号, §3 护城河 降 一档.

11. **销售费用 增速 > 营收 增速**
    - 看: 利润表 销售费用 + 营收 年增长 对比
    - 触发: 销售费用 增幅 > 营收 增幅, 连续 2 年
    - 动作: "增长 靠推力, 不是 拉力", §3 护城河 "品牌 定价权" 打问号.

12. **扣非净利润 / 净利润 < 0.85**
    - 看: 利润表 扣非 数据
    - 触发: 比值 < 0.85, 当年 或 连续
    - 动作: 一次性 非经常 损益 主导 利润, 估值 要 用 扣非 版本, 不 用 报表 净利润.

### K.3 治理 / 股东 类 红旗 (5 项)

13. **关联 交易 规模 异常**
    - 看: 附注 关联方 及 关联 交易
    - 触发: 关联交易 占 营收 > 10%, 或 关联 采购 > 营业 成本 15%
    - 动作: 定价 是否 公允 要 深查; 向 大股东 输送利润 嫌疑.

14. **对外 担保 给 关联方**
    - 看: 附注 对外 担保 明细
    - 触发: 担保 对象 含 大股东 / 关联方, 或 担保 总额 > 净资产 20%
    - 动作: 或有 负债 可能 吞噬 未来 利润, §4.5 红旗.

15. **大股东 占资 / 股权 质押 高**
    - 看: 附注 其他 应收款 + 股权 变动 公告
    - 触发: 大股东 股权 质押 比例 > 50%, 或 其他 应收款 对 大股东 长期 挂账
    - 动作: 股价 下跌 可能 触发 强制 平仓, 引发 治理 动荡; 强红旗.

16. **频繁 并购 / 重组 / 分立**
    - 看: 近 5 年 重大 资产 重组 公告 + 商誉 变动
    - 触发: 3 年内 ≥ 2 次 重大 并购, 或 商誉 年 增量 > 净利润
    - 动作: 管理层 "画大饼" / 资本运作 优先 于 主业 嫌疑; 并购 带来 的 商誉 是 未来 减值 地雷.

17. **管理层 / 大股东 减持 + 送转 组合**
    - 看: 减持 公告 + 分配 预案
    - 触发: 高送转 (10送10 以上) 伴随 大股东 或 管理层 减持
    - 动作: 海润光伏 "高送转陷阱" 样态; 直接 淘汰 观察 名单.

### K.4 审计 / 披露 类 红旗 (3 项)

18. **审计 意见 不是 "标准 无 保留"**
    - 看: 年报 审计报告 正文
    - 触发: 保留 意见 / 无法 表示 意见 / 否定 意见 / 带 实质 内容 的 强调 事项段
    - 动作: 三大前提 ① 直接 判 "假"; 整份 profile 弃权.

19. **审计师 变更 + 异常 留任 过短**
    - 看: 近 5 年 审计师 名称
    - 触发: 3 年 内 变更 ≥ 1 次, 或 新任 审计师 在 关键 争议 前 接手
    - 动作: 深查 变更 原因 公告; 若 原所 与 管理层 冲突 → 强红旗.

20. **管理层 道德 红旗 (历史 虚假陈述 / 违规 处罚)**
    - 看: 证监会 处罚 公告 + 历史 访谈 与 业绩 对照
    - 触发: 历史 有 虚假陈述、内幕 交易、市场 操纵 处罚 记录; 或 连续 3 年 年初 guidance 大幅 高于 实际 (画大饼)
    - 动作: 蚂蚁 / 乐视 样态. 直接 淘汰 观察 名单, 不 进 估值.

## §L. 交易 姿态 & 组合 管理

唐朝 把 "买 / 持 / 卖" 压缩 成 一套 机械 规则, 目的是 剥离 情绪. 本节 提炼 这 套 规则 + 组合 管理 的 具体 约束.

### L.1 买卖 三态 机械 规则

基于 老唐估值法 给出 的 买点 / 卖点 数字, 持仓 姿态 只有 三档 (与 §F 一致, 此处 强化):

| 当前 市值 区间 | 动作 | 细则 |
|---|---|---|
| < 买点 | **加仓 / 建仓** | 买点 = 3y 合理估值 × 50% (高杠杆 × 35%). 分批 进场, 一次 不超过 目标 仓位 的 1/3. |
| 买点 ≤ 当前 ≤ 卖点 | **持有 不动 (收工睡觉)** | 默认 姿态. 期间 每 年 重估 一次 (年报 披露 后), 重大 事件 时 补重估. 不看 股价 波动. |
| > 卖点 | **分批 清仓** | 卖点 = min(3y 合理估值 × 1.5, 当年 净利 × 50 PE). 分 3 批: 触点 卖 1/3; 再 涨 10% 卖 1/3; 再 涨 10% 清仓. |

**重点 澄清** (与 市场 常见 误解 不同):
- "尽可能 不卖" ≠ "永远 持有". 触卖点 必须 卖. 不卖 意味 着 放弃 "把 50 PE 收益 锁定 + 换 更高 性价比 资产" 的 收益.
- 买点 触及 ≠ 必须 立即 满仓. 仍 分批, 留 子弹 应对 进一步 下跌.
- 股价 在 买点 和 卖点 之间 波动 (哪怕 上下 50%) 不 触发 任何 动作. 唐朝 原文 (哲学卷 L6972) "低于 9500 亿 买入, 高于 23000 亿 卖出. 市值 波动 于 9500 亿至 23000 亿 之间, 持股不动. 收工睡觉. 简单吧?" — 这 是 整个 方法论 的 动作 简化 终点.

### L.2 "从 分散 入手, 逐步 集中" — 老唐 的 组合 演化 路径

出处: 投资实战卷 L5382-5400 原文 "第四, 考虑 集中度". 这 是 唐朝 对 "分散 vs. 集中" 争议 的 独特 解. 核心 观点:

```
1. 起步阶段: 极度 分散 = 沪深 300 指数 基金
   理由: 你 什么 都 不 看得懂, 买 指数 = 请 指数 编制方 帮你 淘汰.
   收益 上限 = 市场 平均 (长期 接近 指数 成分股 平均 ROE).

2. 过渡阶段: 指数 + 少量 个股
   理由: 每年 日拱一卒 研究 1 家 企业, 看得懂 一家 就 把 一部分 资金 从 指数
   转到 那家. 看得懂 的 标准 = 能力圈 四问 (§I.3) 全过.

3. 成熟阶段: 集中 持有 4-6 家
   理由: 分散 过度 = 稀释 认知; 集中 过度 = 单一 风险. 4-6 家 平衡 点.
```

**操作 准绳**:
- 单一 持仓 **上限 40%**, 极端 情况下 可 到 50% (老唐 茅台 2017-2020 长期 40%+), 但 必须 满足: 该公司 三大前提 全通过 + 能力圈 四问 全清晰 + 历年 承诺兑现 record 良好.
- 单一 持仓 **下限 约 10%** — 仓位 太低 的 公司, 即使 涨了 也 没 意义; "不敢 重仓 = 没 看懂, 干脆 不 持".
- 持仓 数量 **目标 4-6 家**. 超过 8 家 → 必然 有 几家 没 看懂, 回 指数.
- 行业 集中度: 不 同时 持 > 2 家 同 行业 (避免 行业 风险 集中); 但 允许 同 行业 不同 环节 (e.g. 白酒 高端 + 次高端).

### L.3 现金 管理

唐朝 观点 (投资哲学卷 L10427-10433): 不主动 留 现金 等机会. 只要 有 标的 符合 买点, 应 投入. 不 以 "等 更 便宜" 为由 长期 留 大量 现金.
- 例外: 所有 持仓 都 已 到 合理估值 上方, 且 没有 新 标的 进入 买点 → 此时 自然 产生 现金, 等下 一个 买点 即可.
- 不 主动 择时 卖出 换 现金. "单纯 持有 类现金 资产 等待 股价 下跌 的 所谓 仓位 管理" (哲学卷 L10433) 是 老唐 明确 反对 的.

### L.4 分红 再投 决策

分红 到账 后, 唐朝 的 决策 规则:
1. 现金 分红 到账 → 当前 公司 是否 仍 低于 买点? 是 → 再 买 回原 公司; 否 → 放入 "子弹" 池, 等 下一个 买点.
2. 不 因为 "分红 了 就 必须 立即 投出去" 急于 行动; 也 不 因为 "想 保持 仓位" 就 不 加仓 (这是 机会 成本 思维, 不是 仓位 思维).

**分红率 评判** (见 §E.3 的 总结, 此处 给操作版):
| 组合 | 管理层 评价 |
|---|---|
| 高 ROE + 低 分红率 + 留存收益 再投回报 高 | **优**. 管理层 善用 留存. |
| 高 ROE + 高 分红率 | **良**. 老实, 不乱花. |
| 低 ROE + 高 分红率 | **可接受**. 至少 诚实, 不瞎投. |
| 低 ROE + 低 分红率 + 胡乱 投资 | **劣 ("膀胱效应")**. 留存 毁灭 股东 价值. 强 减分. |

### L.5 下跌 中 的 默认 规则: "不加仓 = 承认 自己 判错"

唐朝 一个 反直觉 的 原则 (散见 于 哲学卷 关于 加仓 讨论): **股价 下跌 后 不加仓, 默认 = 我 承认 自己 原判断 可能 错了**. 加仓 的 前提 是:
1. 重新 做一遍 功课 — 最新 年报 / 季报 + 行业 变化;
2. 确认 企业 基本面 没变, 三大前提 仍 全过, 能力圈 四问 仍 清晰;
3. 当前 市值 确实 处于 新估值 的 买点 以下.

三项 全过 才可 加仓. 否则: 不卖 (也 默认 不 立即 翻案) 但 也 不加. 卖 的 决策 门槛 同样 高: 不 因为 股价 跌 而 卖, 只 因为 "我 发现 原来 的 判断 错了" 而 卖.

这 让 "止损 / 止盈" 这类 技术派 概念 在 唐朝 体系 里 完全 不存在. 卖出 只由 两件 事 触发:
- **估值 逻辑**: 市值 > 卖点 → 分批 清仓.
- **事实 翻案**: 研究 发现 之前 的 判断 错了 (e.g. 新 年报 披露 三大前提 某项 实际 不过, 或 护城河 假设 被 打破).
