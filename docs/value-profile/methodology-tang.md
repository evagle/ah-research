## §A. 唐朝 分析 框架 总览

### A.1 核心 mantra (最高抽象, 贯穿所有分析)

> "**企业的内在价值是企业未来所有自由现金流的折现值之和**" — 格雷厄姆 → 巴菲特 → 老唐 继承

这句话在老唐文集里反复出现。它不是装饰语。它决定了所有后续动作的取舍:
- 不看 "市值 / 股价 / 热度"，只看 "自由现金流的可持续性"。
- 看 PE 时实际看的是 "投资回报率的倒数" (vol2 L2970)。同样期限的债券因还本付息确定性接近, 看 PE 足矣；两家企业之间的 PE 比较"没有意义"，除非先解决三大前提。

### A.2 三大前提 (估值大厦的承重墙) ★★★★★

任何进入估值环节之前，**必须**先对这三个问题给出 "Yes/No + 证据":

```
① 净利润数据是否为真?            (True earnings test)
② 未来能否持续?                   (Durability test)
③ 持续是否需要依赖加大资本投入?   (Free cash flow vs. reinvestment test)
```

出处: vol1 L4309-4315 (2017 茅台 估值文)；vol2 L2980-3001 (长安汽车 对比茅台, 经典阐述)。

**关键应用细节**:
- 三前提 **必须全部满足**。任何一项成 "No"，老唐的 25x / 合理估值法不适用，必须打折或放弃。
- "持续是否需要加大资本投入" = **自由现金流** 的核心。老唐对 "自由现金流" 的定义 (vol2 L6856): **经营所得 − 维持现有盈利能力所需的资本支出**。这不是一般的 "经营现金流 − CapEx"，而是 "只减去 **维持性** CapEx"。
- 典型案例 (vol2 L2988-3001): 甲乙两家公司今年都赚 10 亿。甲能全部分光; 乙只能分一半，另一半必须再投入才能维持明年的 10 亿。同价格，必选甲，因其自由现金流更高。长安汽车报表 PE 7 倍，但为维持销量每年需投 65 亿 CapEx，这 65 亿是 "泡沫利润"，需要 "去泡沫化"。

### A.3 三好 标准 (好生意 / 好公司 / 好价格)

出处: vol1 L12394 (古井贡 分析, 总结为 "一般一般"); vol1 L14663-14665 (雅戈尔 "好公司" 三维度).

| 层级 | 判断对象 | 老唐的检验问题 |
|---|---|---|
| 好生意 | **行业 / 商业模式** | 盈利是否 "真金白银" (高毛利 + 低应收 + 低 CapEx)？客户为什么 必须 买？定价权在哪里？ |
| 好公司 | **管理层 + 护城河** | 管理层历史言行一致吗？分红率 / 股权激励 / 关联交易 是否对小股东友好？ROE 是否持续高位？ |
| 好价格 | **估值 / 市场位置** | 三年后合理估值打 5 折 (非高杠杆) 或 7 折 (高杠杆) 以下? |

三者**顺序不可交换**: 先确认好生意, 再看好公司, 最后算好价格。老唐反复强调 "否决一个就放弃, 不和稀泥"。

### A.4 研究 步骤 (实战流程)

从 vol1 福寿园 "如何从零开始研究一家企业" 系列 (L342-346) 与 雅戈尔 《生人勿近诞生记》(vol2 L15305-15306 "列为必读") 归纳:

1. **招股说明书 + 历年财报** — "首选，不仅权威而且免费" (vol1 L5458)。先读最老一份招股说明书, 看原始产能 / 初始资本 / 最初的 "讲故事" 与现实差异。
2. **资产负债表 简化** — 把资产压缩成 "类现金 / 经营资产 / 生产资产 / 投资资产 / 商誉与无形"。vol2 L10709, L11256, L12912 反复演示 (信立泰): **类现金 61 亿 = 货币 + 银票 + 预付 + 其他**。如果总市值 ≈ 类现金 + 欠条盈余，那市场在白送生意。
3. **利润表 + 现金流量表 交叉验证** — 销售收现 vs. 营业收入 比值, 经营现金流 vs. 净利润 比值, 逐年对比。详见 §D.3.
4. **"去掉财技"** 还原真实销售 — 茅台案例: 报表营收 + (期末预收 − 期初预收)/1.17 (vol1 L1573)。
5. **形成 "三年后合理估值"** — 老唐估值法 (§F).
6. **设买点 + 卖点, 然后收工睡觉** — vol1 L6972: "低于 9500 亿买入, 高于 23000 亿卖出。市值波动于 9500 亿至 23000 亿之间, 持股不动。收工睡觉。简单吧?"

### A.5 心理 / 行为 层 (常被忽略但必讲)

- "**牛市只是投资者的风落之财 (windfall)**" (vol1 L6164) — 收益主要来自土地产出, 不来自傻子馈赠。
- "**尽可能不卖**" (vol1 L6244) — 只有在 "三年后合理估值 150%" **或** "当年 PE 50 倍" (二选其一较低者) 时才触发卖出。默认姿态是 呆坐。
- "**账户里每一毛资本都是自己多年辛苦赚来的**" (vol2 L15452) — 防自欺心态。
- "**下跌的时候, 一个现金流大过净利润, 毛利率高达九成以上的生意, 十五六倍市盈率照样被人当作垃圾一样丢弃**" (vol1 L6701) — 市场先生常态。

## §B. 商业模式 分析 depth

### B.1 核心 商业模式 问题 (不是通用 10 问, 是老唐实际问的)

老唐在每家公司分析里反复追问的 **sharp** 问题 (从 Moutai / 分众 / 洋河 / 海康 案例中归纳):

**关于产品本身**:
1. 产品的 **物理约束** 是什么? (茅台: 酱香 → 空间; 浓香 → 时间。基酒 5 年储存。12987 工艺. vol1 L5650)
2. 产品是 **社交 / 情感 / 收藏 / 功能** 中的哪一类? 各类的需求稳定性不同。茅台是 "社交产品, 强者愈强" (vol1 L4393).
3. **消费者画像**: 真实买单人群是谁? 换一群人还会买吗? (Moutai 反腐 测试, vol1 L6756-6766: 是 "官员爱喝" 还是 "有权者才买得到, 其他群体也有爱"? 决定 反腐 冲击是永久毁灭 还是临时换人。)

**关于定价权**:
4. 公司对客户是 **强势** 还是 **弱势**? (老唐分众 "强" 的三维度: 对客户 / 对上游 / 对手. vol1 L16597)
5. 提价史: 过去 10 年提价频次、幅度, 提价后是否丢客户? (分众 "每年提价 15%", vol1 L16602; Moutai 提价 200 元 毛利率只增 0.8 个点 → 毛利已经很高, vol1 L5691-5693)
6. **出厂价 vs. 终端批价** 的差额。如果差额巨大且持续, 要么公司定价错 (留出空间待调), 要么渠道利润不健康 (将吸引竞争或监管)。Moutai 出厂 969 vs. 一批 2000+ = 老唐 2019-2020 年"三年后必提价" 判断的依据 (vol1 L6820-6889).

**关于竞争 / 商业模式本身**:
7. 有没有 **大、准、强、省、专** 的网络 / 品牌 / 工艺效应? (老唐对 分众 的 5 字 moat 框架, 见 §C.1)
8. 扩大规模能否 **不摊薄毛利**? 边际产能增加的单位成本是上升还是下降?
9. **替代品** 的距离: 不只看当前对手, 更要看 "消费者打发同样需求的替代方案"。(e.g., 电梯广告 vs. 手机广告 vs. 楼下电视 — 老唐倾向于 "互补" 而非 "替代", vol1 L16706).

### B.2 好生意 vs. 坏生意 的具体清单

老唐多次提到的 **好生意** 共同特征 (vol2 L2099-2168 等; 信立泰 / 双汇 / 涪陵榨菜 / 粤高速 等案例):

```
好生意:
  资产结构干净                  (非 "资产 70% 是应收 + 商誉 + 在建工程")
  现金流状态良好                (经营现金流 ≥ 净利润)
  应收账款极少甚至没有          (粤高速 应收 ≈ 0)
  存货管理符合经营需求
  较大数额的预收或应付等经营性占款 (反映议价权)
  销售费用占比低且稳定下降       (反映品牌 pull 而非 push)
  毛利率稳定高位                (= 定价权 + 产品差异化)
  ROE 持续 > 15%                (且不依赖高杠杆)
  分红 "可以分光全部利润"       (不需要持续再投入)

坏生意 (典型信号):
  ROE < 8% 或 波动剧烈           (老唐点杀 门槛: 连续 3 年 ROE > 8%. vol1 L2357)
  毛利率持续下滑
  经营现金流净额 < 净利润        (卖了货收不到钱)
  应收账款 / 营收 > 同业均值
  商誉占净资产比例高
  在建工程长年不转固            (资本支出 "黑洞")
  销售费用增长 > 营收增长        (增长靠推力, 不是拉力)
  "坐庄式" 股权激励 / 定增     (老窖 "无耻定增" vol1 L135)
  管理层发言与业绩数据脱节
```

**点杀标准** (老唐给读者筛股的 "最低门槛", vol1 L2357, vol2 L15531-15549):
```
① IPO > 3 年 (首选 > 5 年, 以获得连续审计数据)
② 最近 3 年 ROE 均 > 8%, 连续 3 年 ROE > 15% 优先
③ 当前 PE < 40 (超过 40 倍意味着需要 3 年利润翻 3-5 倍, 不合理假设)
④ 排除行业: 医药、汽车、地产 (老唐"看不懂")
```

## §C. 护城河 分析

### C.1 老唐 对 护城河 的 分类

老唐 **没有** 构建过巴菲特 / Morningstar 那种 5 分类理论。他分析护城河的方式是 **case-by-case 拆解**。但从案例中可归纳几个反复出现的类型:

| 类型 | 案例 | 关键特征 |
|---|---|---|
| **有效规模** (网络 + 地域自然垄断) | 分众 (梯媒)、白云机场、雅砻江水电、粤高速 | 单终端无价值, 组网后跳跃式增值; 政府不可能批第二家。vol2 L549, L1026 |
| **品牌 + 定价权** | 茅台、五粮液 | 消费者主动买单, 提价不丢客户; 品牌代表 "信息传达 + 时间节约", 对时间宝贵的人价值高 (vol1 L5755) |
| **不可复制工艺 + 时间** | 泸州老窖 (老窖池)、茅台 (茅台镇水源 + 12987) | "再多金钱也无法复制" (vol2 L8003). 时间是护城河的一部分 |
| **管理 + 效率** | 海澜之家、双汇 | 本身商业模式可能不是超 A, 但管理远优于同行, 榨出 "定价以外" 的利润 (vol2 L21904) |
| **社交 + 网络效应** | 腾讯 | "**看不见边际**" (vol1 L204). 社交关系的迁移成本极高 |

### C.2 大、准、强、省、专 — 老唐的 moat 验证框架 (分众案例提炼)

这是老唐最具操作性的 moat 框架 (vol1 L16495, 16521 梳理分众传媒(三)):

```
大   ①规模同行业最大 (给出市占率数字, 如 95% / 70% / 55%)
     ②市场空间还大 (有 headroom, 非天花板)

准   ①受众画像准 (针对性, 能匹配广告主需求)
     ②广告投放准 (转化率 / 触达率 能量化)

强   ①强迫性 / 场景垄断 (消费者无法选择替代渠道)
     ②强势 (对客户 / 对上游 / 对手 都强势 — 三维验证)

省   成本结构碾压对手 (分众不生产内容, 省掉版权 + 明星费用)
     租赁成本 << 内容制作成本

专   专业 / 聚焦 (不是样样都做, 不贪多 — 江南春剥离副业)
     行业最有实战经验的人才 / 模式积累
```

### C.3 如何 **验证** 护城河 (而非 声称)

老唐最常用的 "reality check" 手段:

1. **ROE / 毛利 指标 作为 "路标"** (vol1 L16490-16494): "ROE 指标是个路标, 它的作用是指引我们去寻找那些不在报表里的资产。分众 ROE 维持在 70%, 一定有重要的资产不在资产负债表里。"

2. **提价能力测试**: 历史上每次提价，客户流失吗? 销量涨吗? (Moutai: 8 次提价, 每次销量都涨; 分众: 年提 15%, 客户数年增.)

3. **对手 测试**: 同行业 top 对手盈利能力如何? 若行业 second-best 都在亏损, 说明 leader 的 moat 远超账面。(分众 L16641: 新潮传媒 "规模远大于任何物业或业委会, 净利润依然为负" → 证明分众的 "组网价值" 真实)

4. **反向验证 (消费者 必须)**: 切换成本具体是什么? 可不可以举一个 "我是客户, 我为什么不换" 的具体场景?

5. **"组网价值" 增值 是否能被 leader 独占?** (vol1 L16631): "这份增值由组网者建立, 当然应该由组网者享受"。关键问题: 如果组网价值跳跃式, 但进入壁垒低, 新对手也能快速成网, leader 仍无 moat.

### C.4 ROE 杜邦 分解

老唐对 ROE 杜邦 的态度 **务实而保守**:
- 他不会细到逐年 杜邦 三项画折线 — 他看的是 **"连续 3-5 年 ROE 是否稳定在 15%+"**.
- 他关心的杜邦细节: **净利率 (反映定价权)、总资产周转率 (反映运营效率)、财务杠杆 (反映风险)**. 若 ROE 高但杠杆也高 (银行、保险), 需要对三前提做特别审查 — "贷/存比" "拨备覆盖率" 等。
- 他反复强调: **高 ROE 的秘密通常不在杜邦三项里, 而在 "未被计入账本的资产"** (e.g. 茅台镇水源、分众的终端网络、腾讯的社交链). 杜邦是起点, 不是终点。
- 他会做一个 **逆算 ROE**: "如果茅台把 1000 亿现金分掉, 净资产收益率会高到惊人, PB 也会大到惊人" (vol1 L5591). 这暗示财务结构里"沉淀冗余净资产"会稀释 ROE, 并非企业真实盈利能力。

## §D. 财报 排雷 细节

### D.1 老唐 最 强调 的 红旗 signals (按严重性排序)

来自 vol1 獐子岛 / 雅戈尔 / 乐视网 / 海润光伏 / 蚂蚁 等 "避雷要诀" 系列案例:

**致命红旗 (看到立刻删除观察名单)**:
1. **生物资产**: "农、林、渔、牧 = 造假高危区" (vol2 L15287). 獐子岛 扇贝"跑了" 两次.
2. **商誉/净资产 > 20%**. 后续任何年度都可能一次性计提, 直接干掉利润。
3. **关联交易 + 股权投资 + 炒股收益 占净利润 > 30%** (雅戈尔案例). 投资者实际上买了一个 "炒股账户", 而非实业。
4. **现金流量表负的 / 大幅小于净利润**, 且持续多年 (非短期行业周期). 表示利润是纸面的, 收不回现金.
5. **管理层道德红旗** (vol2 L21452): "管理团队的历史决策所依据的道德观和价值观, 绝对是我们排除企业的重要依据之一。这或许可能让我们错过牛股, 但同样会保护我们避免掉进一些可怕的陷阱。" (老唐 "不喜欢马云" → 避开蚂蚁暂缓上市陷阱. vol2 L21446)

**重要红旗 (减分项)**:
6. 预付账款 / 其他应收款 超常增长 (vol2 L871, 联美控股 10 亿其他应收款 = 控股股东借款). 这些科目是 "大股东占款 / 关联方输血" 的经典伪装点.
7. 在建工程 长年不转固定资产. 扣折旧/摊销就是利润, 不转固就是 "把成本挂起来不确认" — 伪装利润.
8. **销售费用 大幅增长 > 营收增长** (增长靠推力).
9. 应收账款 / 营收 比值 持续上升 (先卖货, 收款越来越慢 → 渠道压货).
10. **高送转 + 减持** 组合 (海润光伏 "高送转陷阱" vol2 L15418).

### D.2 特定 科目 分析 清单 (老唐实战查点)

| 科目 | 老唐 问的 具体 问题 |
|---|---|
| **货币资金** | 和 "有息负债" 比, 如果账上大把现金却还有大量有息负债 → 现金疑似虚假 / 受限 / 被挪用 |
| **应收账款** | 增幅 > 营收增幅? 账龄结构? 前 5 大客户集中度? 坏账计提比例 是否激进/保守? |
| **应收票据** | 银票 vs. 商票占比? 商票违约风险. Moutai 案例: 应收票据 飙升往往对应 销售政策放松 (vol1 L1746) |
| **预付账款** | 给谁预付的? 是否常年挂账? (关联方通道?) |
| **其他应收款** | **查关联方**. 数额大 + 挂账长 = 重大疑点 (vol2 L871 联美控股 案例) |
| **存货** | 跌价准备 计提充分? 存货周转率 vs. 同业? 生物资产 / 高价原料 → 造假区 |
| **在建工程** | 总额 vs. 固定资产 比例. 长期不转固 = 挂账操纵折旧 |
| **商誉** | **商誉/净资产 > 20% 视为雷区**. 每年减值测试披露看假设 (revenue growth / WACC) 是否激进 |
| **预收账款 / 合同负债** | **正向指标** (客户提前付钱 = 议价权). Moutai 把它作为调节报表工具, 需要 "还原" (vol1 L1573) |
| **其他流动资产** | 主要是 银行理财, 视为 类现金 |
| **可供出售金融资产 / 交易性金融资产** | "浮亏"或"浮盈"要核实, 不能信账面. (雅戈尔 中信股份浮亏 案例 vol2 L15330) |

### D.3 现金流 真伪 验证 方法 (老唐最犀利的武器)

出处: vol2 L21548-21560, 以 Moutai 为例的推导:

```
销售收现 = 营业收入 + 增值税 (17%) 
         - 应收账款增加额 
         - 应收票据增加额 
         - 贴现应收票据付出的财务费用 
         + 预收账款增加额
```

**应用方法**: 对目标公司, 用这个公式算出 "应该收到多少现金", 然后和现金流量表里 **"销售商品、提供服务收到的现金"** 逐年对比。
- 若 实际 > 公式值 → 现金流真实, 可能还有额外回款;
- 若 实际 显著 < 公式值 → 有 "纸面营收" 嫌疑, 进一步查应收 + 票据结构.

**还原真实销售** (排除预收款操纵): vol1 L1573-1574:
```
剔除预收款后的真实销售 = 营业收入 + (期末预收款 − 期初预收款) / 1.17
```
Moutai 案例: 2016 年报表营收 389 亿, 预收款净增 92.8 亿, 还原后真实销售 = 389 + 92.8/1.17 = 468 亿 (vol1 L4362).

**经营现金流 vs. 净利润**: 老唐的经验值
- "经营现金流 ≥ 净利润" 是及格线;
- **"经营现金流 >> 净利润"** (如 2 倍, 说明利润被折旧/摊销低估) = 优质企业信号 (vol2 L491, L8957);
- **"经营现金流 < 50% 净利润"** 持续 2 年以上 = 红旗.

## §E. 管理层 评估

### E.1 看年报的哪些部分识别管理层品质

1. **"致股东信"/ 管理层讨论与分析** — 语气是否自大 / 自省? 是否直面坏消息? (Moutai 袁仁国 2014 "3% 增长" 预告, 实为 "被迫", vol1 L749)
2. **业绩预告 vs. 实际** — 历年偏差方向和幅度. Moutai 2009-2013 五年预算报告 vs. 实际完成 (vol1 L981) 是老唐经常用的信心 vs. 兑现 对照表.
3. **信息披露质量** — 产量 / 销量 / 库存 / 产品结构细节是否披露 (vol1 L6438-6443: Moutai 李保芳 任内 "信息披露质量翻天覆地").
4. **分红率 + 股权激励** — 见 §E.3.
5. **关联交易** — 规模, 定价机制, 公众股东否决权. (Moutai 章程 第 79 条, vol1 L5486)
6. **"膀胱效应"** — 本应分给股东的利润被管理层胡乱投资 (vol1 L1317).

### E.2 言行一致 的 verification 方法

这是老唐最强调的。核心操作:
1. **找历史访谈 / 新闻稿 / 业绩发布会** 里管理层的 "承诺" 或 "判断" — 扩产进度、提价计划、新业务目标、行业展望。
2. **建一张 Excel** (老唐 自己就这么做的): 每年管理层承诺 vs. 年底实际达成. 差异大的年份标红. 系统性乐观 / 系统性保守, 都是 signal.
3. **历史多年 的轨迹 决定可信度** — 不看单次, 看连续 5-10 年模式。李保芳 "允许系列酒政策性亏损三年" → 事后 "10 倍营收" 兑现 (vol1 L6476-6491), 所以老唐给李董打高分.
4. **承诺 vs. 激励** 一致性: 股权激励方案的业绩考核线, 是否 贴合 管理层公开目标? 太低 (送钱) or 太高 (无法兑现) 都是红旗.

### E.3 历史 预测 vs. 兑现 的 tracking 操作 (老唐最具体的工具)

Moutai 管理层评估 case study (vol1 L975-988):
```
表 2: 茅台 管理层 预算报告 vs. 实际 (2009-2013)
  2011 信心满满 → 现金流指标显示 未来强增长 → 2011 实际超预算
  2012 信心膨胀 → 动用 预收款 强行营收 +44% → 为 2013 埋雷
  2013 信心碰壁 → 反腐+限额 → 前高后低, 曲线从"持续成长"扭曲为"日落西山"
```
**教训**: 管理层信心周期与业绩周期同步, 顶峰时乐观, 拐点时不认. 要用**多年窗口**看, 不看单年。

**分红率 评判** (vol1 L1300-1317, 回复 赵丹阳 关于分红的讨论):
- 分红率 **不是** 越高越好, 要结合 ROE 看.
- "挣的多 + 分的少" (ROE 高 + 分红率低) = 管理层把股东的钱再投入高回报项目, **好**.
- "挣的少 + 分的多" (ROE 低 + 分红率高) = 没有好项目, 把钱发回股东, **也好, 至少诚实**.
- "挣的多 + 分的少 + 再投入回报低" = **烂管理层** (膀胱效应).
- 判断再投入回报的方法: 历年 留存收益 ÷ 净利润增长 比率.

## §F. 老唐估值法

### F.1 公式 derivation

**两句话**: "三年后 15~25 倍 PE 卖出能赚一倍的价格, 就可以买入。高杠杆企业打七折。" (vol2 L2985-2986)

**完整步骤** (vol1 L6921, Moutai 2020.4.27 演示):

```
Step 1: 预估 "3 年后 的 归母 净利润"
        — 分业务 板块 拆解 (Moutai: 茅台酒 + 系列酒 + 财务公司)
        — 对每块做 量 × 价 × 净利率
        — 给一个 上下限 (保守 vs. 乐观) → 取中值 ± 10%

Step 2: 给 合理 PE 倍数
        合理 PE = 1 / 无风险收益率
        无风险收益率 3%~4% → 合理 PE 25-30 倍 (vol1 L10317, L5810)
        对于 "三大前提 都 确认 + 自由现金流 > 利润" 的 顶级企业, 取上限 30 倍
        一般企业 取 25 倍
        **重点: 取值与企业利润增速无关, 只与无风险收益率相关** (vol1 L5810)

Step 3: 3 年后 合理估值 = 3 年后净利润 × 合理 PE
        以 Moutai 为例 (2020.4.27 计算):
        三项合计 628-788 亿 净利 → 700 亿 ± 10%
        合理 PE 27.5 (取 25-30 中值)
        → 合理估值 = 700 × 27.5 × ± 10% ≈ 19000 ± 10% 亿

Step 4: 买点 = 3 年后合理估值 × 50%
        = 19000 / 2 = 9500 亿 (Moutai 2020.4.27 老唐买点)
        高杠杆企业 打 7 折, 即 × 35%

Step 5: 卖点 = min(3 年后合理估值 × 150%, 当年净利 × 50 PE)
        ① 3 年后合理估值 × 110%(安全边际) × 150% = 31350 亿
        ② 2020 年估计净利 460 亿 × 50 PE = 23000 亿
        取较低: 23000 亿 = 卖点
        分批卖: 达 23000 亿卖 1/3; 再涨 10% 卖 1/3; 再涨 10% 清仓 (vol1 L7045)

Step 6: 持仓区间 = [买点, 卖点]
        此区间内 持股不动, 收工睡觉.
```

### F.2 参数 选择 细节

**无风险收益率**: 老唐取 5 年期国债 (vol2 L2983) 或 当前市场 10 年期国债收益率。2013-2020 多年间他用 3.5%-4%。2020 后有所下调 (低利率环境)。
- 若 利率 上升到 5%, 合理 PE 应降到 20; 利率下降到 2%, 合理 PE 可到 50.
- 他反复警告: **"老唐估值法, 仅适用于符合三大前提的企业"** (vol1 L6184). 对不符合的垃圾股, "举着火把穿越炸药库", 估值无意义。

**50% 打折 (买点 = 合理估值 × 50%)** 的逻辑:
- 三年赚 100% → 年化 26%. 若无法承受估值误差, 选此.
- 高杠杆企业打 7 折 (即 35%) — 因为三前提 的"持续是否依赖资本投入"更脆弱.

**150% 溢价 (卖点 = 合理估值 × 150%)** 的逻辑:
- vol1 L6314-6323: 假设企业 25% 增长 + 4% 无风险收益率. 50 PE 卖出后, 3 年无风险 = 本金 × 1.125. 3 年后企业净利 × 1.95 × 25 PE = 新价格。用新价重买, 能多买 ~15% 股数.
- 若回落早于 3 年或增长低于 25%, 占便宜更多.

## §G. Moutai specific insights ★ (seed for Part 1 deepening)

### G.1 老唐 对 Moutai moat 的 unique framing

**超越 "品牌" 的 4 维 moat** — 老唐把 Moutai 护城河拆得远比券商研报细:

1. **空间的稀缺性 (地理 moat)** — 茅台镇 7.5km² 核心产区 + 中华村扩产区. 酱香不能异地复制 (vol1 L5650). 基酒年产天花板 5.6-7 万吨 (vol1 L5570: "保芳董 公开说过 5.6 万吨后不再考虑扩产"). **这是最硬的 moat — 无法被资本 / 技术复制**.

2. **时间的稀缺性 (工艺 moat)** — 12987 (1 年周期 + 9 次蒸煮 + 8 次摊凉 + 7 次取酒). 基酒储存 5 年. 每年基酒挥发约 2% (vol1 L5712, 季克良 披露). 这决定了 **产销 与 4-5 年前 基酒产量 强相关** — "四年前基酒产量 × 75%-85% = 当年可售商品酒数量" (vol1 L5745).

3. **社交 / 收藏 需求** (非 "喝" 需求) — 老唐明确拒绝 "收藏不是真实需求" 的说法 (vol1 L5644): "存新酒喝老酒, 将会是长期存在的需求". 这层需求 **与 GDP 强相关, 但与健康趋势弱相关**.

4. **消费者扩散 moat** — "强者愈强" (vol1 L4393): 喝的人越多, 越容易被选择 (社交证明). 反腐 不能毁掉茅台, 因为"官员"不是群体属性, 只是"获取渠道"属性 (vol1 L6762).

### G.2 跟踪的 关键 数据 (老唐 Moutai 季度/年度 excel)

从 vol1 整理出 老唐 tracks 的 Moutai 核心数据:

```
产能 / 供应侧 (决定 3-5 年后 可售量):
  □ 当年基酒产量 (吨, 包括茅台基酒 + 系列酒基酒 分列)

  □ 储存空间 (旧酒库 + 新酒库)

销售 / 需求侧:
  □ 当年茅台酒商品酒销量 (吨)
  □ 当年系列酒销量 (吨) + 营收
  □ 出厂均价 (综合均价, 不只是普茅 969)
  □ 直销比例 (直销价 ≈ 1499, 是普茅 出厂价的 1.55×, 占比提升 = 均价提升)
  □ 一批价 (经销商批发价, 核心健康指标)
  □ 一批价 / 出厂价 差额 (差额大 → 未来提价空间 ∝ 此差额)
  □ 经销商数量变化 (整顿 → 直销转化)

财务 操纵 / 真实度:
  □ 预收账款 / 合同负债 (季末余额, vs. 营收的比率)
  □ 应收票据 余额 + 结构 (银票 vs. 商票)
  □ 销售收现 / 营业收入 比值 (应 > 1 + 增值税率)
  □ 销售费用 绝对值 和 占营收比 (茅台正常 4-5%, 异常时需深查)
  □ 分红率 和 绝对分红金额 (期望 每股 17.5 元+)

产品结构:
  □ 普茅 vs. 非普茅 (小批量 / 陈年 / 纪念) 占比
  □ 系列酒: 王子 / 迎宾 / 汉酱 / 仁酒 各自销量 + 营收 + 毛利
```

### H.2 具体 可 copy 到 SKILL.md Step 3b 的 prompt 改进 (3 条 highest-ROI)

以下 3 条直接可以注入 section-worker 的 dispatch 模板:

#### 改进 A — §1 商业模式 / §3 护城河 共同注入 "三大前提 checklist"

```
在输出 §1 / §3 的 分析之前, 必须先 独立回答以下三个问题 (每项 [真/假/存疑] + 证据):

① 净利润数据 是否 为真?
   证据来源: 经营现金流 vs. 净利润 比值 (最近 3 年), 销售收现 vs. 营业收入 (应 > 1 + 增值税率),
           应收账款 / 营收 变化, 存货 周转率.
   
② 未来 能否 持续?
   证据来源: 需求基础 (消费者画像 + 切换成本), 产能瓶颈, 行业竞争格局,
           2-3 个最硬的替代品 / 竞品 分析.

③ 持续 是否 需要 加大 资本 投入?
   证据来源: 最近 5 年 CapEx / 折旧摊销 比值, 在建工程 / 固定资产 趋势,
           ROIC (不 是 ROE) 稳定 vs 下降.

如果 任一项 = "假" 或 "存疑", 则本节 置信度 必须 为 "低", 并在 §6 估值 模块 被拒绝.
```

#### 改进 B — §3 护城河 写作 prompt 重写

```
写 §3 护城河 时禁止使用以下空话: "具有强大品牌", "具备护城河", "技术领先", "管理优秀".
必须完成以下 5 步结构:

1. **分类**: 公司的护城河 属于 以下哪一类? (多选):
   □ 有效规模 / 自然垄断   □ 品牌 + 定价权   □ 工艺 + 时间 不可复制
   □ 网络 / 社交效应       □ 管理 + 效率     □ 其他 (specify)

2. **老唐 "大、准、强、省、专" 框架** 应用 (若 适用):
   □ 大 — 规模 同业 最大? 市场 空间 还大? 给出 市占率 数字
   □ 准 — 用户 / 场景 匹配 精度? 量化 指标?
   □ 强 — 对客户 / 对上游 / 对 对手 各自 强势 证据?
   □ 省 — 相对 对手 / 替代 渠道, 成本结构 碾压?
   □ 专 — 聚焦, 不贪多; 管理层 是否 剥离 副业?

3. **验证 (而非声称)**: 至少 给出 2 条 "reality check":
   □ 历史 提价 测试 — 提价后 客户 流失 还是 留存?
   □ 对手 测试 — 同行业 top 对手 盈利 如何? (second-best 盈利差 强化 leader)
   □ 切换成本 测试 — 一个 具体 场景: "我 作为 客户, 为什么 不换?"
   □ 组网 价值 独占 — leader 跳跃增值 能否 被 独占?
   □ ROE 路标 — 高 ROE 的 "看不见的 资产" 在哪?

4. **定量 痕迹**: 最近 5 年 毛利率 / 净利率 / ROE (非杜邦 三项 分解, 而是 稳定性) 
   + "经营 现金流 ÷ 净利润 比值"(应 ≥ 1).

5. **反方 观点**: 列 至少 1 个 "会打破 这个 moat 的 情景" (技术变化 / 消费偏好 / 监管 / 对手 变招).
   不回答 的 护城河 分析 = 空话 rejected.
```

#### 改进 C — §6 估值 模块 注入 "老唐估值法 三数字" 必填

```
§6 估值 输出 必须 包含 以下 7 个 字段 (任 一 缺失 视为 unfinished):

[1] 三前提 判断: ① [真/假/存疑] ② [...] ③ [...]
    (任一 非 "真" → [2][3][4] 必须 附加 additional discount factor)

[2] 3 年后 归母 净利润 预估: [下限] — [中值] — [上限], 
    拆分 业务板块 (至少 2 块), 每块列 量 × 价 × 净利率 计算.

[3] 合理 PE = 1 / 当前 无风险 收益率 
    当前 10y 国债 收益率 = __%
    合理 PE = __ (保守边 25, 乐观边 30; 超出区间需 justify)

[4] 3 年后 合理 估值 中枢 = [2] × [3]
    范围 = 中枢 ± 10%

[5] **买点** = 3 年后 合理 估值 × 50%
    (高杠杆 企业 × 35%; 且必须 说明 为什么 判定 高杠杆)

[6] **卖点** = min(3 年后 合理估值 × 150%, 当年 净利 × 50 PE)
    当年 净利 估算 = ___ (简短说明来源)
    two candidate 都 列, 取 较 低者.

[7] **持仓 姿态**: 
    □ 当前 市值 < 买点 → "买入建仓 / 加仓"
    □ 买点 ≤ 当前 市值 ≤ 卖点 → "持有 不动 (收工睡觉)"
    □ 当前 市值 > 卖点 → "分批 清仓 (1/3 at 卖点, 1/3 at 卖点+10%, 1/3 at 卖点+20%)"
    → 具体 仓位 建议 必须 与 [1] 三前提 判断 一致 (任 一 为 "假" 严禁 建仓 / 持有)
```
