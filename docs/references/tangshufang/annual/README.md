- `../03-个股复盘案例.md` — 茅台/腾讯/分众/海康/洋河/平安+招行

## 跨年度观察轴（供查阅时作索引）

- **持仓演化**：茅台 / 腾讯 / 分众 / 洋河 / 海康 / 古井贡 是跨年核心名单；陕西煤业 2019→2022 从 0 建到 6%、2022 清仓；美团 2021→2023 建到清；福寿园 2020→2023 建到清
- **估值公式演化**：2016 公开 → 2018 确立 → 2019 系统化（含杠杆打折）→ 2020 仪式化（50% 买点 / 150% + 50PE 卖点）
- **卖点首次实战**：2021-01-08 茅台 2149 + 分众 11.26 同日触发
- **熊市心态**：2018 首次亏损年清仓重构；2022 熊市纪律执行；2023 滚动 3 年首次负收益
- **监管事件**：2020 蚂蚁上市前夜暂停 → 2021 互联网监管 → 2023 游戏管理办法草案
- **年度收益**（自报）：2016 未披露 | 2018 -17.8% | 2019 +52% | 2020 +66% | 2021 -4% (基金净值法 -3.84%) | 2022 近 -0% (V型反转) | 2023 -7%

## 使用建议

研究某只股票时，若涉及老唐买卖过的标的，建议先查对应年份的"持仓变化"章节，再去 `../03-个股复盘案例.md` 看跨年总结。研究估值方法、心态、行业判断则先看主题式文件。
