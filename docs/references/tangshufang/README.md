# 唐书房方法论参考（外部学习材料）

本目录为**外部学习材料**，归档来自唐朝（唐书房）投资体系的方法论提炼，供本项目 value-profile skill 作为理论参考。

## 来源

- 《老唐合集：投资哲学》(Lao Tang Collection Investment Philosophy)
- 《老唐合集：投资实务》(Lao Tang Collection Investment)

## 定位

本目录是**只读参考**。本项目自己的分析框架和操作规则, 以及 value-profile skill 运行时用到的 reference 文件, 位于 `.claude/skills/value-profile/`（`SKILL.md` + `template-zh.md` + `references/`），那些文件是项目自己的声音, 不再标注 "唐朝 / 老唐" attribution。

如需追溯某个原则的出处或想看原始案例, 翻这里。
