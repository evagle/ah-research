**综合判定**:
- **0触发** → 大股东无侵占, 小股东利益与大股东完全一致
- **1-2项 低程度触发** → 轻度抽血, 不影响投资论点, 但需监控演化
- **≥ 3项触发 或 任一"中高程度"触发** → §4.6 置信度降一档, 估值折扣加深5-10%
- **≥ 5项触发 或 任一"严重"触发** (如 > 10% 不公允关联交易 / 资金占款规模 > 净资产5% / 大股东质押连环担保) → §2.12.2 **一票否决**, profile 终止

**国企特殊提示** (控股股东为国资委/地方政府/省国资):
抽血渠道可能包括以下"合规抽血"形态——不是违规但对 small shareholder 是机会成本:
- **政策性任务**: 上市公司承担非经济效益的社会责任 (扶贫/就业/维稳/地方产业配套)
- **地方政府强制采购**: 要求使用特定本地供应商, 定价高于市场
- **税收/土地优惠 vs 反哺**: 看似给优惠, 实则要求配套投资/慈善捐赠
- **商标与品牌权属**: 母公司集团持有核心商标, 上市公司每年付使用费
- **集团销售公司分成**: 销售环节通过集团下属销售公司, 吃中间差价
- **公益与基建摊派**: 地方性基建/公益项目强制配套出资

**茅台案例** (启发本节设计): 贵州茅台 的 控股股东 茅台集团 (贵州省国资委 100% 持股) 历年通过以下渠道 低强度抽血:
1. 商标 "茅台" 归 集团, 上市公司 支付年度 使用费
2. 集团销售公司分成 非上市主体 吃 加价差
3. 地方性 慈善/公益 承诺 (支持贵州 教育/扶贫/文化)
4. 省政府配套基建 (机场/水库/道路) 摊派 投资

对茅台这种超强护城河+高 ROE 生意, 这些渠道"无伤大雅"(年度合计 < 净利润3%, 估值影响有限), **但若是差生意+同样抽血, 则小股东几乎无剩余利润**。因此: **护城河强度 > 抽血强度 是唯一救赎**。
