# Valuation — 本项目估值法与买卖决策参考

本文件定义**本项目估值法**: 公式、适用边界、买卖触发。运行 skill 的 Step 6时直接引用。

---

## 1. 核心公式

### 1.1合理估值
```
合理估值 = 3 年后归母净利 × min(1 / rf, 25PE) × (1 ± 10%)
```
- **rf**: 10年期国债收益率, 近年 ~3.5%, 对应 PE 锚 ~28x
- **上限25PE**: 即使 rf 极低, PE 也不超过25（避免长期利率假设崩溃）
- **±10%**: 合理估值区间而非单点

### 1.2买点
```
买点 = 合理估值 × 50%  (标准情形)
买点 = 合理估值 × 35%  (高杠杆 / 周期 / 不确定性高)
```
高杠杆定义: 有息负债/净资产 > 1, 或 有息负债 / CFO 近3年 > 3。

### 1.3卖点（双轨, 取较低者）
```
卖点轨道 A = 合理估值 × 150%
卖点轨道 B = 当年净利 × 50PE
卖点 = min(A, B)
```
A 轨保护在极端乐观情境下的退出；B 轨保护在未来利润 significant slippage 后退出。

### 1.4 "股权债券"视角 (Equity Bond, §1.1的经济直觉)

PE 锚的背后是一个"把股票看成利息不固定的债券"的思维框架, 用来判定股价便宜贵时当参考心锚:

```
初始收益率 = 税前 EPS / 股价
```

- **摸索判断**: 初始收益率 > 长期国债利率 × 2 (e.g. 当前 rf ~3.5%, 对应阈值 ~7%) = 有吸引力候选
- **换算对照**: 初始收益率7% ⇔ PE ~14 (税前); 5% ⇔ PE ~20; 4% ⇔ PE ~25 (本项目上限)
- **护城河公司独有优势**: 息票 (EPS) 会随时间增长, 不像真债券利息固定; 持有10年的真实年化收益 = 初始收益率 + 复合增长率

**何时用**: §2三档 NI 测算做完后, 用 Equity Bond 视角做一次 **心算交叉检验** —— 若中性档3y NI 对应初始收益率明显低于长期国债 ×1.5, 说明当前合理估值 / PE 锚**太乐观**, 应 reconsider 是否把 PE 锚从25降到20或更低。

**不替代 §1.1公式**: 这是一把心锚/直觉工具, 不替代 `合理估值 = 3y NI × PE 锚` 主公式; 两者得出的合理价差异 > 30% 时, 取较保守者, 并在 profile 里注明理由。

## 2. 三年后净利润预估

### 2.1必填三档
乐观/中性/悲观, 至少2个业务板块拆解。每档结构:
```
业务板块 i 营收_t+3 = 量_t+3 × 价_t+3
业务板块 i 净利润_t+3 = 营收_t+3 × 净利率_t+3
净利润_t+3 = Σ 业务板块 i 净利润_t+3
```

### 2.2三大来源
1. **管理层指引** (优先, 若存在): 年报董事会报告、业绩说明会、投资者交流会的量化预期
2. **行业天花板 + 份额**: 行业总量天花板 × 目标份额 × 行业净利率
3. **产品线外推**: 现有产品线的 CAGR 假设, 新品/新市场/并购 独立估

### 2.3每档必附 "具体假设一句"
不接受 "我们认为未来3年会增长"。必须:
```
悲观: 主营产品 A 价稳定, 量年化 +2%, 净利率降至 30% → 归母净利 XXX 亿
中性: 产品 A 提价 3%/年, 量年化 +5%, 净利率 33% → XXX 亿
乐观: 提价 5%/年, 量年化 +8%, 新产品 B 贡献 5 亿利润, 净利率 35% → XXX 亿
```

## 3. 6类生意估值矩阵

对每家公司先归类, 再套对应的估值方法。**同时符合多类 → 取最严档**。

| 生意类型 | 估值方法 | 买点 (合理估值 ×) | 卖点 | 关键约束 |
|---|---|---|---|---|
| **强护城河消费/互联网龙头** | PE 锚 = min(1/rf, 25), × (1 ± 10%) | 50% | min(150%, 50PE) | 三大前提必过; 护城河宽/中 |
| **周期股** | 穿越周期平均利润 × 15PE, 波峰打7-8折 | 40-50% | 繁荣期 PE 到10即减仓 | 用周期平均, 不用单年; 禁用当前净利 |
| **高成长股（年化 > 25%）** | 25PE 上限不破 | 50% | 50PE 必减 | 不用 PEG 抬到40+; 保守下限; 反向需3年窗口 |
| **金融 — 银行** | PB (1.0-1.15 × 真实净资产) | 35% | PB > 1.3× 真实净资产 | 真实净资产 = 账面 − 未计提不良 |
| **金融 — 保险** | 默认回避 | — | — | EV 折现假设多且无法客观验证 |
| **高杠杆 (地产/部分电力)** | PE 下调到8-12 | 35% | PE > 15减仓 | 硬指标: 有息负债/净资产 > 1 |
| **公用事业 (水电/高速/港口)** | 稳态 FCF / (rf + 2%) | 股息率 > rf × 1.3 | 股息率 < rf 退出 | 看折旧 vs 维持性 CapEx 差额 |

### 3.1 "不适用 PE" / "默认回避"
- 保险/精细化工/重周期钢铁/寿险/创新药/猪肉养殖/生物资产重 → Step 6不输出估值数字, Part 0标 "定性研究 only"
- 理由: 要么假设过多无法客观验证, 要么资产端不可查

### 3.2分档不确定时
- 多类叠加, e.g. 高杠杆 + 周期 → 取最严档（高杠杆的8-12PE + 35% 买点）
- 护城河 窄/弱 → 自动降一档 PE 上限（25 → 20）

## 4. 适用性边界

PE 锚估值法**不适用**于:
- 当年或下年净利润为负的公司（无 PE 可算）
- 净利润构成中非经常性损益 > 30%（E.g. 卖子公司/政府补贴大）
- 资本结构剧变期（e.g. 刚 IPO 或 刚重大并购）
- 会计准则切换当年（金融资产重分类/租赁准则切换/收入准则切换）

这些情况下, Step 6只输出定性判断, 不给数字。

## 5. 持仓姿态三态 + 分批规则

### 5.1三态
- **低于买点**: 分批建仓, 3-5批, 每批间隔5-10% 降幅触发
- **买点到卖点之间**: 持有, 不加不减（"价格不是信息" — 详见 discipline.md）
- **高于卖点**: 分批减仓, 3-5批, 每批间隔10-15% 升幅触发

### 5.2分批操作
- 单次不超过总仓位的20%
- 连续3次同方向操作后暂停1个月, 避免情绪化操作
- 新 ticker 首次建仓不超过组合的5%, 验证2个季度后才允许加到目标权重

## 6. 估值动摇的处理

估值判断在研究过程中 "微调" 是正常的, 但若:
- 3年后净利润中性档需要下调 > 20%
- PE 锚需要从25降到15
- 三大前提从 "过" 变 "不过"

这叫 **估值动摇**, 不是微调。动摇发生时:
1. 停止买入/继续 buy（即使价格已跌到原买点之下也不买）
2. 重走 §1 / §3 / §5, 重新判定三大前提
3. 若重判后估值结论改变 → 更新买点卖点, 继续
4. 若重判后发现当初判断错了 → 认错, 按新估值决策（可能是清仓）

详见 `discipline.md`。

---

## 7. 全稀释股本调整

计算归母净利时:
- **加** 可转债若全部转股后的股本稀释
- **加** 管理层已授予未行权的期权/激励股
- **不加** 股份回购（已回购库存股应从总股本中扣除）

3年后的每股收益应按全稀释股本计算, 避免 "表面 EPS 高, 实际稀释后一般" 的陷阱。

**置信度**: 高 / 中 / 低
**置信度低的原因**: ...
