# Discipline — 本项目的操作纪律与心态参考

本文件是本项目 value-profile skill 在面对 "买 / 卖 / 持 / 认错" 决策时的纪律手册。估值的数字是输入, 本文件是操作的输出。

---

## 1. 估值动摇即停手

### 1.1 定义
"估值动摇" 不是估值微调, 而是:
- 3 年后中性档净利需下调 > 20%, 或
- PE 锚需要从 25 降到 15, 或
- 三大前提 (行业 / 护城河 / 管理层) 从 "过" 变为 "不过", 或
- 发现关键数据是错的 (e.g. 市占率数字、管理层历史、财报数字)

满足任一即 "动摇", 非 "调整"。

### 1.2 动摇后的处理流程
1. **立即停止买入**。即使股价已跌到原买点之下, 不要 "越跌越买"。
2. **重走 §1 / §3 / §5**: 行业 / 护城河 / 管理层的判断重新做一遍。
3. **若重判后结论仍然成立, 只是参数变了** → 更新买点卖点, 继续持有并继续分批（若仍低于新买点）。
4. **若重判后发现当初判断错了** → 认错, 按新估值决策（可能是继续持有, 可能是减仓, 可能是清仓）。
5. **若重判过程中发现管理层道德问题 / 财务造假嫌疑** → 直接清仓, 不再复议。

### 1.3 为什么不能 "越跌越买"
"越跌越买" 是基于原估值继续有效的假设。若估值动摇, 原买点已失效, 此时买入是加仓错误判断。正确的姿态是先停手, 重判完再决策。

## 2. 认错 > 坚持

### 2.1 "判断动摇" vs "市场情绪"
- **判断动摇**: 来自新的基本面信息（年报数据 / 管理层言行 / 行业政策 / 竞争格局变化）。必须认错。
- **市场情绪**: 来自股价波动 / 分析师评级 / 新闻情绪 / 宏观恐慌。忽略。

区分标准: **若下跌 / 上涨的原因可以用一句话讲清楚背后的基本面变化, 且那个变化会改变 3 年后净利润或护城河判断, 就是判断动摇**。否则是市场情绪。

### 2.2 认错的操作
1. 在 profile 里 Part 0 写 "本次 profile 的判断调整: [动摇原因] + [新结论]"
2. 若结论从 "值得买" 变为 "不值得" → 按新结论决策, 不管当前浮亏浮盈
3. 认错不丢人, **隐瞒错误才丢人**。早认错亏小, 晚认错亏大

### 2.3 "坚持" 的正确用法
"坚持" 只在以下情境使用:
- 估值未动摇, 但股价波动剧烈 → 坚持原计划分批建仓 / 持有
- 三大前提仍然成立, 但短期业绩不及预期 → 若短期业绩对 3 年后判断无实质影响, 坚持

坚持 ≠ 面子。面子是情绪反应, 不是决策依据。

## 3. 市场先生条款

### 3.1 三条铁律
1. **波动不是信息**: 每日涨跌只反映交易对手情绪, 不反映公司价值变化。一家公司的价值在一个月内不会因为股价变动而变动。
2. **价格不改变价值**: 合理估值是 "3 年后净利 × PE 锚", 与当前股价无关。股价跌不会让价值变低, 只是让买入更合算。
3. **不从价格反推观点**: 涨了找 "为什么涨" 的逻辑 / 跌了找 "我是不是错了" 的逻辑, 是反向推理, 禁用。判断必须独立于价格。

### 3.2 什么时候看价格
- 建仓 / 加仓决策: 看是否低于买点, 低多少, 决定批次
- 减仓 / 清仓决策: 看是否高于卖点
- 估值动摇后: 重判之前先停手, 价格无关

**其他情境都不应该看价格**, 包括:
- 查研究进度时
- 读年报时
- 做护城河分析时
- 和同行讨论时

### 3.3 价格信息的使用规则
股价 = 参考事件, 不是 input:
- 股价新高 → 问 "是否已到卖点", 不问 "为什么涨"
- 股价新低 → 问 "是否已到买点", 不问 "为什么跌"
- 股价跳空 → 问 "是否有估值动摇的基本面事件", 不问 "接下来会怎么走"

## 4. 耐心的操作化

### 4.1 空仓不是成本, 是能力
- 找不到值得买的, 空仓。**空仓比持有烂公司强**。
- "手里有钱 = 必须买" 是散户思维。正确姿态: 手里有钱, 等机会。
- 一年出手 1-2 次很正常, 一年不出手也正常。

### 4.2 机会成本的正确理解
- **错误理解**: "我空仓半年损失了 10% 的机会成本, 因此我必须找点什么买"
- **正确理解**: "错过的机会的 ex-ante 成本是 0, 因为事前我不知道它会涨。我只为 ex-post 的真实决策负责"

### 4.3 什么时候空仓
- 所有研究过的标的都 >= 卖点
- 手里持仓全部被迫清仓（e.g. 估值动摇 + 管理层道德问题）且尚未找到新标的
- 宏观极端不确定性（但这种情境罕见, 别滥用）

### 4.4 空仓的风险
- 长期空仓有滑入 "择时" 陷阱的风险。空仓后仍要继续研究扩大标的池, 等好标的出现。
- 空仓期间本金存余额宝 / 短债基金, 不存活期（活期的机会成本是真实的 3% / 年）。

## 5. 卖出的三触发

仅在以下任一情境触发卖出, 其他情境一律持有:

### 5.1 估值高估 (卖点触发)
- 股价 > min(合理估值 × 150%, 当年净利 × 50PE)
- 分批减仓 3-5 批, 每批间隔 10-15% 升幅触发
- 减仓期间若股价回落至卖点之下但 > 合理估值, 暂停减仓但不回购

### 5.2 买入理由消失
- 三大前提中任一 "不过" (行业 / 护城河 / 管理层)
- 管理层道德红旗（虚假陈述 / 违规处罚 / 股东利益输送 / 关联方大额资金占用）
- 审计意见从 "标准无保留" 降为保留 / 否定 / 拒绝
- 核心产品被替代或核心市场消失

### 5.3 发现更好标的
- 新标的预期 3 年后收益率显著高于现有持仓（> 30% 差距）
- 新标的三大前提都过, 现有持仓某项已在边缘
- 组合优化需要腾出仓位

**非触发**: "涨太快了" / "亏了很多" / "别人赚了" / "分析师下调" — 都不是卖出触发。

## 6. 持仓权重管理

### 6.1 基本规则
- **单一上限 25%**: 单只股票占组合不超过 25%, 避免集中度过高
- **4-6 家深度**: 组合聚焦 4-6 家公司, 每家研究深度达到 profile 要求
- **不做 "为了分散而分散"**: 10 家以上的分散组合不是组合管理, 是懒惰
- **同行业不超 2 家**: 避免行业风险集中, 但允许同行业不同环节

### 6.2 分红再投资
- 分红到账后, 在 30 天内重新分配至:
  - 仍低于买点的持仓 → 加仓
  - 买卖点之间的持仓 → 按当前权重加仓
  - 高于卖点的持仓 → 不加, 留现金等机会
- **禁止**直接再投资同一标的（若该标的已 >= 合理估值）

### 6.3 新标的建仓规则
- 首次建仓不超过组合 5%, 2 个季度后才允许加到目标权重
- 目标权重 = 对 profile 确信度 × 潜在年化收益率 / 组合加权基准
- 从 5% 到目标权重的过程分 3-5 批, 每批对应一个关键验证事件（如年报验证 / 指引兑现）

## 7. 常见情境的决策

| 情境 | 错误反应 | 正确反应 |
|---|---|---|
| 股价一天跌 10% | 慌 / 抄底 / 卖出 | 不看价格。问 "今天有无估值动摇事件" |
| 股价一天涨 10% | 追高 / 卖出 | 问 "是否到卖点"; 否则持有 |
| 三季报净利下滑 | 立即卖出 | 问 "是否改变 3 年后判断"; 否则持有 |
| 年报披露管理层有刑事调查 | 观望 | 清仓。不复议 |
| 行业政策出台 | 情绪性反应 | 重走 §2 行业判断 + §3 护城河 |
| 竞对推出颠覆性新品 | 情绪性反应 | 重走 §3 护城河 + 悲观情景 |
| 分析师一致下调评级 | 跟风卖 | 忽略。分析师的判断不是你的输入 |
| 连续 3 年表现差 | 再等一年 | 重走全流程。3 年是重判触发点 |

## 8. 每次操作前的自检清单

下单前强制回答以下 5 个问题, 任一答不出来 → 暂停操作:

1. **我当前的判断是什么** (三大前提 / 估值锚 / 买卖点)?
2. **相对上一次操作, 判断有变化吗? 变化依据是什么**?
3. **如果明天股价再跌 20% / 涨 20%, 我的行动是什么**?
4. **我承认错了会怎么做**?
5. **我的操作是基于新信息还是情绪**?

若第 5 问的答案是 "情绪", 关屏幕, 24 小时后再看。
