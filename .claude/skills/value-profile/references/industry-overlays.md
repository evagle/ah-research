# Industry Overlays — 按行业的 KPI 与判读参考

本文件为本项目 value-profile skill 在不同行业场景下的 KPI overlay 和阈值表。用于 §1-§3 的行业 context 阶段和 Step 5 的排雷阶段。

---

## 1. 白酒

### 1.1 关键 KPI
| 指标 | 定义 | 健康区间 | 红旗 |
|---|---|---|---|
| 批价 | 终端批发价 | 稳定或温和上涨 | 短期跌破出厂价 |
| 出厂价 | 酒厂到经销商 | 每年可提价 3-5% 是强 | 连续 2 年未提价 |
| 批价 / 出厂价 | 渠道利润空间 | ≥ 1.2× | < 1.1× 经销商积极性差 |
| 基酒产能 | 决定 4 年后高端产能 | 扩产 vs 高端需求匹配 | 扩产 > 需求 50% |
| 经销商动销率 | 到货周转天数 | 30-60 天 | > 90 天渠道库存积压 |
| 高端 vs 次高端 结构 | 千元价位带占比 | 龙头应 ≥ 30% | 占比下滑 |
| 渠道库存 | 行业 + 公司级 | 1-2 月出厂量 | > 3 月需警惕压货 |

### 1.2 白酒生意特点
- **4 年基酒规律**: 今年销售的酒大多是 4 年前的基酒, 扩产到出货有 4 年滞后
- **提价权测试**: 龙头每年能顺利提价 → 强护城河, 卡在批价附近无法提价 → 次高端天花板
- **消费场景**: 商务宴请 / 礼品 / 自饮, 前两者受宏观 + 反腐 / 公务影响大
- **地区属性**: 省酒 (区域龙头) vs 全国化品牌, 前者抗周期差

### 1.3 估值锚
- 高端酒龙头 (强 / 宽护城河) → 25PE 标准锚
- 次高端 → 20-25PE, 看全国化进度
- 地方酒 → 15-20PE, 谨慎

## 2. 银行

### 2.1 关键 KPI
| 指标 | 定义 | 健康区间 | 红旗 |
|---|---|---|---|
| ROA | 资产回报率 | 零售行 ≥ 0.9% / 对公行 ≥ 0.7% | < 0.5% |
| 不良率 | NPL ratio | ≤ 1.5% | > 2% 或快速上升 |
| 拨备覆盖率 | 拨备 / 不良 | 250-400% | < 150% 或 > 500% (藏利润) |
| 净息差 | NIM | 2.0-2.5% | < 1.8% 或波动剧烈 |
| 资本充足率 | CAR | ≥ 12% | 接近监管红线 10.5% |
| 核心一级资本 | CET1 | ≥ 8.5% | < 8% 需募资 |

### 2.2 真实净资产计算
```
真实净资产 = 账面净资产 − 未计提不良真实损失
未计提不良 ≈ 关注类 × 30% + 逾期 90天+ × 50% + 展期类 × 70%
```
做完这一步, 再看 PB。许多 "0.5 PB" 实际上真实 PB 是 0.9。

### 2.3 银行双类商业模式
- **存贷利差类** (大多数国有 + 城商行): 拉存款 / 放贷款, 看息差 + 不良
- **零售平台类** (少数行): 手续费 + 财富管理 + 信用卡, 看 AUM + 客户数 + 非息收入占比

后者更接近消费品逻辑, 可给更高 PB; 前者周期性强, PB 天花板 1.15。

### 2.4 估值锚
- 零售强 / AUM 模式 → 1.0-1.3 PB
- 国有大行 → 0.7-0.9 PB, 股息率 > 6% 是好价格
- 城商 / 农商 → 看地方经济, 谨慎, 0.6-0.8 PB

## 3. 互联网平台

### 3.1 关键 KPI
| 指标 | 定义 | 解读 |
|---|---|---|
| DAU/MAU | 日活 / 月活 | 粘性, > 0.4 高粘性 |
| ARPU | 人均贡献收入 | 分业务看, 游戏 > 电商 > 广告 |
| 变现节奏 | 从 MAU 到 Revenue 的漏斗 | 漏斗衰减率 |
| 投资性损益占净利 | 公允价值变动 + 投资收益 | > 30% 说明主业赚钱能力失焦 |
| 广告 vs 电商 vs 游戏 拆分 | 各业务营收 / 毛利 | 识别真正的现金牛和烧钱业务 |
| 现金流对比 | 经营 CFO / 自由现金流 | 是否真金白银 |

### 3.2 互联网判读要点
- **变现集中度**: 80% 利润来自 1 个业务线 → 抗风险差
- **用户获取成本 CAC / 终身价值 LTV**: LTV / CAC > 3 才算健康
- **平台 vs 自营**: 平台模式轻资产高毛利 (30-50%), 自营模式重资产低毛利 (5-15%)
- **监管敏感度**: 游戏 / 金融科技 / 在线教育 历史教训, 政策灰犀牛要看清
- **"投资收益" 剔除**: 看经营性净利润, 把二级市场持仓波动剥离

### 3.3 估值锚
- 现金牛业务稳定, 估经营性净利 → 20-25PE
- 广告 + 游戏混合 → 15-20PE（监管折扣）
- 纯成长股（盈亏平衡前）→ 用 PS, 参考同业

## 4. 消费品（非白酒）

### 4.1 关键 KPI
| 指标 | 健康区间 | 红旗 |
|---|---|---|
| 渠道库存天数 | 30-60 天 | > 90 天 |
| 动销 | 月月正增长或稳定 | 连续 2 月下滑 |
| 新品成功率 | 新品 3 年占比 ≥ 15% | 新品失败率 > 80% |
| 广告费 / 销售额 | 5-15% | > 20% 靠营销硬推 |
| 经销商数量 / 覆盖 | 稳定增长 | 数量减少但没换血 |
| 毛利率 | 强品牌 ≥ 40% | 持续下滑 |

### 4.2 消费品判读
- **渠道库存 vs 动销背离**: 公司发货数 >> 终端销售数 → 压货, 未来必反噬
- **大单品依赖**: 单品贡献利润 > 50% → 脆弱性
- **品类天花板**: 行业总量增速 < GDP → 只能靠份额争夺
- **渠道变迁**: 传统商超 → 电商 → 社区团购 → 直播, 迁移期龙头可能失位

### 4.3 估值锚
- 强品牌 + 宽护城河 (e.g. 调味品 / 乳业龙头) → 20-25PE
- 品牌力中 → 15-20PE
- 新品牌 / 品类天花板低 → 10-15PE 或 PS

## 5. 资源 / 周期股

### 5.1 关键 KPI
| 指标 | 解读 |
|---|---|
| 穿越周期平均利润 | 5-10 年净利中位数, 而非单年 |
| 资本开支 vs 折旧 | CapEx / 折旧 > 1.5× 扩张期, < 1 维持期 |
| 现金成本曲线位置 | 成本前 25%（低成本）/ 中位 / 后 25%（高成本） |
| 储量 / 产能 | 可开采年限 > 15 年较稳 |
| 期货 / 现货 套保比例 | 高套保 = 利润锁定, 低套保 = 裸多头 |

### 5.2 周期股特有
- **禁用当前净利估值**: 必须用穿越周期平均, 波峰期估值要打 7-8 折
- **"反身性陷阱"**: 高景气下, PE 自然下降, 看起来 "便宜", 实际上波峰后利润 60% 蒸发
- **成本曲线**: 龙头在前 25% 低成本档 → 全行业亏损时仍能赚钱, 是真护城河
- **资本开支周期**: 从投建到产能释放 3-5 年, 判断新产能冲击要提前

### 5.3 估值锚
- 低成本龙头 → 穿越周期平均利润 × 15PE, 波峰 10-12PE
- 中位成本 → 穿越周期平均利润 × 10-12PE
- 高成本 → 默认回避

## 6. 高杠杆地产

### 6.1 关键 KPI
| 指标 | 健康线 | 红旗 |
|---|---|---|
| 已锁定未结转收入 | 1.5-2 年营收 | < 0.5 年 |
| 土储货值 | 3-5 年销售 | < 1.5 年补库存压力 |
| 有息负债 / 净资产 | ≤ 1 | > 1.5 |
| 现金短债比 | > 1.2 | < 1 流动性风险 |
| 经营活动 CFO | 正 | 连续 2 年负 |
| 三道红线 | 全绿 | 1+ 档红档 |

### 6.2 地产判读
- **土储 + 已锁定 + 销售 = 收入预估**: 已售未结转 ≈ 未来 1-2 年收入
- **利润滞后**: 今年卖的房, 2-3 年后才结转, 因此净利润是 2-3 年前销售决定
- **杠杆敏感性**: 销售下滑 20% + 融资受限 = 链式崩盘, 不存在 "稍微差一点"
- **开发 vs 持有**: 开发型 ROE 高但风险大, 持有型 (商业 / 物业) 现金流稳

### 6.3 估值锚
- 开发型 → PE 8-12, 买点合理估值 35%
- 持有型 (REITs 式) → 租金收益率 + 折现, 看股息率
- 三道红线黄 / 橙 / 红档 → 默认回避

## 7. 公用事业

### 7.1 关键 KPI
| 指标 | 健康线 | 红旗 |
|---|---|---|
| 稳态 FCF | 近 5 年平均自由现金流 | 波动 > 30% |
| 维持性 CapEx | ≤ 折旧 1.2× | > 折旧 2× (实际是扩张) |
| 分红率 | ≥ 50% 且稳定 | 波动剧烈 |
| 股息率 | > rf × 1.3 | < rf |
| 总装机 / 吞吐量 | 稳定或缓慢增长 | 下滑 |
| 电价 / 过路费 | 政策调整记录 | 频繁下调 |

### 7.2 公用事业判读
- **类债券属性**: 稳态现金流 + 高分红 + 低成长 = 债券替代, 但比债券多一份抗通胀
- **折现率 = rf + 2%**: 因为现金流确定性接近债券但有增长空间
- **监管敏感**: 电价 / 收费标准由政府定, 监管从紧会压缩利润
- **重资产 + 长寿命**: 折旧期 20-40 年, 维持性 CapEx 远低于折旧 → 自由现金流 > 账面净利

### 7.3 估值锚
- 稳态 FCF / (rf + 2%) = 合理市值
- 若股息率 > rf × 1.3 → 好价格
- 若股息率 < rf → 退出（不如直接买债券）

## 8. 行业不做清单

以下行业本项目**默认回避**, 不纳入 profile 覆盖范围:

| 行业 | 回避理由 |
|---|---|
| **保险（寿险）** | EV 折现假设（投资回报 / 死亡率 / 退保率）多且无法客观验证, 内含价值不可查 |
| **创新药** | 成功率 < 10%, 估值靠管线假设, 不具可重复性 |
| **精细化工** | 产品 SKU 数百种, 毛利率分散无法聚合分析 |
| **重周期钢铁 / 煤炭 (非龙头)** | 成本曲线后 75% 位置, 周期下行亏到破产 |
| **猪肉养殖 / 水产** | 生物资产, 可走动会飞, 审计困难（獐子岛 / 雏鹰） |
| **影视 / 游戏 (单品爆款驱动型)** | 单品成败决定, 无可重复护城河 |
| **壳公司 / 借壳 / 频繁重组** | 主营难辨, 财报不可信 |
| **小市值 (< 50 亿) 公司** | 流动性差, 即便对也可能卖不出价 |

遇到以上行业, Part 0 标注 "行业在本项目回避清单", 不再继续 profile 流程。

### 例外
- 以上行业若出现龙头级别的 "特例"（e.g. 一家技术绝对垄断 + 现金流极好的创新药公司）, 可以做**定性研究**, 但不给估值数字。
