# Financial Reading — 本项目财报阅读操作参考

本文件是本项目 value-profile skill 的财报深读手册。核心目的: **用财报排除企业**, 而非用财报挖掘企业。

---

## 1. 阅读顺序

年报 500+ 页, 按下列顺序读, 发现红旗就立即止步:

1. **封面 + 目录** — 审计机构 / 是否变更 / 报告期
2. **重要提示 + 风险因素** — 管理层承认的问题在哪
3. **董事会报告** — 管理层对年度经营的自评
4. **经营业绩讨论** — 产品线 / 区域 / 渠道分解, 与年初指引对照
5. **资产负债表** — 结构是否异常
6. **现金流量表** — 利润是否变成现金
7. **利润表** — 收入结构、费用率、净利率
8. **附注** — 红旗的真正藏身处
9. **审计报告** — 意见类型

**为什么先读资产负债表**: 它是利润表和现金流的 "输出累积", 且作假难度大（每笔科目都要交叉钩稽）。利润表最容易作假, 因此放在最后对照读。

## 2. 资产负债表 — 5 分类 + 金融资产新准则

### 2.1 五分法
- **生产经营资产**: 固定资产 / 无形资产 / 在建工程 / 存货 — 赚钱的生产力
- **投资资产**: 长期股权投资 / 金融资产 / 投资性房地产 — 赚其他钱
- **经营周转资产**: 货币资金（含受限）/ 应收账款 / 预付账款 — 业务周转
- **应付经营资产**: 应付账款 / 预收款项（含合同负债）/ 应付工资 — 别人欠你的
- **融资相关**: 有息负债 / 股东权益 — 资本结构

### 2.2 金融资产新准则（2018-2020 切换后）
- **以摊余成本计量** — 俗称 "瘫子", 不入当期损益
- **以公允价值计量且变动计入当期损益** — 俗称 "笋子", **直接吃利润表**, 最危险
- **以公允价值计量且变动计入其他综合收益（债权类）** — "菜粽", 入 OCI 可转出
- **以公允价值计量且变动计入其他综合收益（权益类指定）** — "肉粽", 入 OCI 终生不可转出

**读法**: 凡是 "笋子" 金额大的, 利润波动主要来自估值变动而非经营, 降级处理。

## 3. 必读附注 (高优先级 12 项)

每份 profile §2.11.2 强制覆盖以下附注。遗漏 ≥ 3 项视为不合格, 退回重读。

| # | 附注 | 看什么 | 红旗 |
|---|---|---|---|
| 1 | 货币资金 | 受限金额 / 保证金 / 冻结 | 受限占比 > 30% |
| 2 | 应收账款 | 5 大客户名 + 账龄结构 + 坏账计提 | 单一客户 > 30%, 或 1 年以上占比 > 20% |
| 3 | 应收票据 | 银票 vs 商票比例 | 商票占比 > 50%（收款质量差） |
| 4 | 预付账款 | 对象 + 用途 | 对关联方预付大额 |
| 5 | 其他应收款 | 关联方占比 | 关联方占比 > 50% |
| 6 | 存货 | 分项（原材料 / 在产品 / 库存商品）+ 跌价准备 | 库存商品增速 > 营收增速 ≥ 20pp |
| 7 | 在建工程 | 项目 / 年限 / 转固计划 | > 3 年仍未转固, 或占比 > 总资产 20% |
| 8 | 商誉 | 并购对象 / 减值测试假设 | 商誉 / 净资产 > 30% |
| 9 | 合同负债 | 占营收比 + 同业对标 | 低于行业均值 50% 可疑, 高出 2× 可能提前确认收入 |
| 10 | 应付账款 | 议价权信号（账期 vs 同业） | 账期明显短于行业均值 |
| 11 | 长投 + 其他权益投资 | 被投单位有无造假 / 经营异常 | 被投亏损但账面不减值 |
| 12 | 有息负债 | 结构 / 利率 / 到期日 | "大存大贷"（同时有巨额现金和巨额有息负债） |

## 4. 红旗清单（简版 — 高危模式）

遇到以下任一模式, 立即降级为 "观察" 或直接剔除:

1. **商誉 / 净资产 > 30%** — 一次减值能让净资产腰斩
2. **其他应收款 / 净资产 > 10%** — 资金被挪用或输送的高发区
3. **在建工程 > 3 年不转固** — 要么工程失败, 要么规避折旧
4. **经营现金流 / 净利润 < 50% 连续 2 年** — 利润真实性红旗, 可能是应收膨胀
5. **管理层道德红旗** — 历史虚假陈述 / 处罚 / 关联交易非正常价 / 股东利益输送 → 直接大幅降级, 不再讨论生意好坏
6. **大存大贷** — 账上有几十亿现金 + 几十亿有息负债 + 仍在借款, 99% 现金造假
7. **生物资产 / 农林渔牧** — 獐子岛式可走动 / 会飞的资产, 审计困难
8. **应收账款增速 > 营收增速 ≥ 20pp 连续 2 年** — 销售乏力, 靠放账填营收

## 5. 三表勾稽公式（4 条必查）

### 5.1 真实营收
```
真实营收 ≈ 营收 + Δ合同负债 − Δ应收账款 − Δ应收票据 (商票部分)
```
若真实营收 << 报表营收, 利润被提前确认。

### 5.2 销售收现
```
销售收现比 = 销售商品 / 提供劳务收到的现金 / 营业收入 (含税)
```
健康值 ≥ 1.05（因为营收不含税, 收现含税）。< 1 连续 2 年, 应收账款或合同负债异常。

### 5.3 净利润 → 经营现金流桥
```
CFO = 净利润 + 折旧摊销 − Δ应收 − Δ存货 + Δ应付 + Δ合同负债 + 其他非现金
```
走一遍桥, 每一项都要有合理解释。不平的部分就是可疑。

### 5.4 维持性 CapEx 近似
```
维持性 CapEx ≈ 折旧摊销 × (1 + 通胀系数)
自由现金流 ≈ CFO − 维持性 CapEx
```
若 CFO > 总 CapEx 长期成立, 属 "省" 类生意。

## 6. 造假识别的一般模式

不特指哪家公司, 列出高频作假手法:

- **收入端**: 关联方购销 / 突击发货 / 提前确认工程完工 / 放宽信用（填放账到应收）
- **成本端**: 在建工程停止转固（少折旧）/ 存货不跌价 / 研发资本化比例异常高
- **现金端**: 期末短暂借款补现金 / 受限资金混入非受限 / 理财产品分类为 "现金等价物"
- **利润端**: 卖资产 / 政府补助 / 公允价值变动 / 一次性所得税返还 凑利润
- **结构端**: 频繁并购制造商誉 → 后续慢性减值 / 子公司并表时点选择 / 关联方剥离亏损业务

对每家公司跑一遍以上 5 个维度, 列 "本公司是否沾边 + 页码引用 + 金额"。

## 7. 读年报顺序（快速扫描模式）

当 3 年内要读 ≥ 10 家公司时, 使用此快速扫描:

1. **封面 (1 min)** — 审计机构 / 意见 / 是否变更
2. **董事会报告 (5 min)** — 管理层自评 + 下年度计划
3. **重要事项 (5 min)** — 诉讼 / 处罚 / 担保 / 关联交易
4. **财务摘要 (5 min)** — 3-5 年关键指标 (营收 / 利润 / ROE / CFO)
5. **附注中的 12 项 (30 min)** — §3 的表
6. **管理层薪酬 + 持股变动 (5 min)** — 是否套现
7. **关联交易披露 (10 min)** — 金额 / 定价公允性

总计 ~1 hour / 家。若任一步发现红旗, 该公司立即进入 "排除" 清单, 不再继续。

---

## 8. 对 skill 运行的约束

- 每个财报引用必须带 **年报页码**。"管理层讨论说盈利能力强" 无页码 → 退回。
- 数字引用格式: `(2024 年报 p.XX)` 或 `(招股书 p.XX)`。
- 跨年对比必须连续, 缺一年 → 说明原因（e.g. 2020 年 IPO 前数据不可得）。
- 同一事项 3 个以上来源交叉验证（年报 + 公告 + 券商研报 + 监管披露）。
- 禁止用 "据了解 / 据分析师 / 据说" 作为财报事实来源。

## 9. 特殊场景的加读清单

| 场景 | 必加读的附注或章节 |
|---|---|
| 首次覆盖 | 近 3 年所有重大合同 + 管理层历史 + 股东背景 |
| 近 1 年变更审计机构 | 变更公告 + 前任审计师辞任声明 + 新任意见类型 |
| 商誉占比 > 20% | 每项并购的减值测试假设（折现率 / 永续增长率）|
| 有息负债 / 净资产 > 1 | 到期结构 + 再融资计划 + 利息保障倍数 |
| 金融资产 > 营业收入 | 笋子 / 菜粽 / 肉粽 分类 + 历史波动 |
| 行业有政策风险 | 法规引用 + 管理层应对 + 同业表态 |
