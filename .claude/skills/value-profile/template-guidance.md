# Template Guidance — 各 section 指导问题 + 数据源 hint

本文件存放 `template-zh.md` 各 section 的完整 `<!-- 指导问题 -->` / `<!-- 数据源 hint -->` / `<!-- 写作规范 -->`。lean template-zh.md 只留 heading + `**本节目标:**` 一行 + 填写区 + `**引用:**` / `**置信度:**` / `**管理层口径校核:**` 字段; subagent 填写时按需 grep 本文件取该 section 的指导。

> **使用方式**: subagent 填某 section 时, 先 Read template-zh.md 找到 section heading + 本节目标; 然后 grep 本文件 `### §X.Y` anchor 取详细指导问题与数据源。

## Part 0 执行摘要 填写指导

- heading 只用一层 `### 执行摘要`, 不另套 `### 结论速览` 二级。
- 每项 bullet 分层: 状态行（结论）+ 3-5 个 sub-bullet 核心证据, 每个 sub-bullet 1 句浓缩。**不必压到 1 行**。目标 1-2 分钟读完。
- 英文缩写强制中文化: CFO → 经营现金流 / NI → 净利润 / CapEx → 资本开支 / FCF → 自由现金流 / EBITDA → 息税折旧摊销前利润 / TAM-SAM-SOM → 潜在-可服务-实际市场规模。仅 ROE / PE / PB / CAGR 等业内普及缩写保留。
- 禁写 `XXX (§x.y)` 样 inline self-reference——事实本身自证, 不指向源 section。细节在 Part 1-5 里。
- 跟踪项 / 风险项 用 `⚠️ **跟踪 N**:` 或 `⚠️ **注意**:` 格式突出, reader 一眼识别。
- 结论用自然中文状态词: 管理层用 **诚信合规** 不用"无触发"; 排雷用 **零触发项** 不用"无触发"。

## Part 1 — 定性分析

> 定性 > 定量。本 Part 以年报 + 招股说明书为主要数据源, 辅以行业研报 + 财经新闻做管理层口径的交叉校核。

## §1 商业模式

> 核心是广义产品。企业以何种方式提供何种产品, 满足何种客户何种需求, 如何盈利。

### §1.1 公司核心资产、主营产品和服务

指导问题:
- 产品怎么生产: 原材料 / 采购流程 / 生产流程 / 工艺技术 / 半成品 / 成品 (数据源: 招股说明书"业务与技术"、研报)
- 公司的核心资产是什么, 能否继续扩张, 扩张策略和空间? 举例: 腾讯 = 游戏和微信 QQ 用户; 茅台 = 基酒; 分众 = 媒体点位
- 主营产品及分产品 / 行业收入占比?
- 单位经济学: 单位成本 vs 单位收益。举例: 茅台单瓶成本 & 收入; 分众单点位成本 & 收入
- 产品的生命周期、市场占比和成长空间?
- 关键资源和关键环节: 研发 / 销售 / 品牌 / 供应链 / 物流 等。举例: 苹果 = 主设计研发销售, 制造外包
- 利润的决定变量是什么 (量 / 价 / 成本 / 周转), 是否能预测? 举例: 茅台 = 基酒产量; 造纸 = 纸浆价格; 水电 = 电费价格
- 行业和公司应该关注的最重要指标是什么? 变化情况如何?
- 原材料的供需情况、公司的议价能力?
- 产品 / 服务的市场情况, 是否有定价权?
- 业务属于高利润率 / 高周转率 / 高杠杆的哪一种类型?

数据源: 年报"主营业务分行业分产品"; 年报"经营情况讨论与分析"; 招股说明书"业务与技术"; 雪球 F10

### §1.2 公司客户

指导问题:
- 客户的地理区域、人口统计特征 (年龄 / 性别 / 收入 / 教育)?
- 面向 2C / 2B / 2G, 各占比?
- 对大客户的依赖程度? **大客户依赖的 4 个坏处** (不宜长期投资):
  1. 业绩预测性差 — 大订单不是天天有, 单一大客户流失业绩一落千丈 (例: 果链公司被踢出业绩一落千丈)
  2. 企业定价权弱 — 大客户更有话语权
  3. 大订单不代表业绩 — 可能随时取消
  4. 客户集中风险 — 单一客户依赖是长期投资禁忌
- 客户粘性如何? 是否可以随意找到替代品? 更换产品难度如何? 是否占领了客户心智?
- 客户具有哪些特征? 客户群体本身面临什么变化? 目标客户长期变化格局?

数据源: 年报"经营情况讨论与分析"; 年报"重要事项"（大客户）; 招股说明书"业务与技术"

### §1.3 差异化

指导问题:
- 替客户解决了什么问题?
- 与竞品相比有何特别之处? 差异化按**复制难度从易到难** (非严格) 分 7 类:
  1. **价格差异化**: 低价策略 / 差异化定价 / 价值定价 (高端定价体现溢价)
  2. **产品 / 功能差异化**: 独特功能 / 口味 / 服务 / 性能 / 质量 / 创新设计 (例: 茅台酒)
  3. **服务 / 体验差异化**: 优质客户服务 / 个性化体验 / 快速响应 / 便利性 (例: Airbnb、好游戏)
  4. **渠道差异化**: 独特分销渠道 / 专属合作伙伴 / 更广更深的触达能力
  5. **技术 / 专利差异化**: 独家技术 / 专利 / 难复制能力壁垒 (但有时效性)
  6. **品牌差异化**: 独特品牌形象 / 故事 / 情感联结 (信誉 / 信任 / 知名度)
  7. **生态 / 平台差异化**: 构建平台 / 网络效应 / 合作伙伴生态锁定用户
- 竞争对手能否提供同等产品或体验? 为什么没能?
- 为什么没抄: 不愿 / 不能 / 不屑?

数据源: 年报"核心竞争力"; 招股说明书"核心竞争力"; 行业研报

### §1.4 盈利模式

指导问题:
- 靠什么赚钱? 盈利来源可能包括:
  - 生产并销售产品或提供服务
  - 赚产品差价 (例: 沃尔玛赚差价)
  - 中介费
  - 订阅或会员费 (例: Costco 赚的是会员费)
  - 广告费
  - 租赁费
  - 专利授权费
  - 加盟费
  - 品牌溢价 (例: 茅台)
- 靠有形资产还是无形资产赚钱?
- 是否需要大额资本开支维持和扩张? (现金流量表"购建固定资产、无形资产和其他长期资产支付的现金" 含维持 + 扩张, 需结合商业模式和在建工程详情区分)
- 直营 vs 加盟?
- 轻资产 vs 重资产?
- 成本结构 / 费用结构 / 收入结构?

数据源: 年报"财务报告"（利润表/现金流量表）; 年报"经营情况讨论与分析"

### §1.5 生意特性

指导问题:

**好生意特点 (完整清单)**:
- 需求长期存在而且刚性、成瘾性
- 差异化竞争
- 弱周期
- 轻资产
- 迭代慢
- 维持当前规模和扩张不需要高资本开支

**产品特性**:
- 是否有差异性, 能否长期维持? (没有产品差异性 = 不可能有护城河, 不投)
- 品牌价值 = 差异化的浓缩 (例: 诺基亚 = 质量好耐摔; 苹果 = 系统流畅生态好; 谷歌 = 最好的搜索服务)
- 品牌差异化不一定全在产品本身, 也可能在服务、独特产品体验 (情感差异化)
- 品牌价值长期通过持续提供优秀且一致的产品和服务来保证
- 相对于竞品的优势, 为什么竞品不抄
- 是否有垄断特性 (全国垄断 / 区域垄断)
- 产品的可替代性

**经济特性**:
- 顾客类型: 2C / 2B / 2G
- 价格高低、定价权
- 购买频次、产品保质期
- 社交和情感价值、成瘾性
- 技术迭代周期、产品生命周期
- 标准化程度、可复制性和边际成本
- 同质化 vs 差异化
- 轻资产 vs 重资产
- 周期性 (原因、周期、阶段)
- 进入门槛 (外部竞争者携巨资是否能抢占市场)
- 维持和扩张需要的资本开支
- 公司吸引力: 增长弹性和潜力大 vs 业绩稳定性和确定性高
- 未来公司主要靠什么业务推动发展? 业务结构将发生什么变化? 对业绩的影响?

**避免买这些特性的生意**:

高风险 — 排除:
- 高负债 / 靠借钱经营 (资本结构风险)
- 现金流极差 / 全是应收 (赚纸钱)
- 产品无差异 / 薄利竞争 (难形成护城河)
- 依赖单一大客户或供应商 (集中风险)
- 监管 / 补贴依赖 (政策一变即受创)
- 热衷并购且商誉高 (整合失败风险, 并购企业缺乏内生增长)
- 高资本开支且回收慢的行业

中风险 — 需谨慎:
- 高技术迭代但研发投入不足
- 贸易型 / "两头在外" 企业 (除非有规模 / 网络优势)
- 高库存敏感型业务 (除非有优秀库存管理)
- 高员工流动 / 依赖关键人 (通过管理层稳健性评估)

数据源: 年报"管理层讨论"; 行业研报

### §1.6 现金流状况

指导问题:
- 公司是否赚钱, 赚的是不是真金白银 (销售收现率 / 净现比)?
- 赚钱能否持续?
- 维持当前盈利所需的资本开支有多大?

**从生产 / 销售 / 运营三环节分析** (结合定量结果统一得出现金流状态):
- **生产过程**: 投资环节的资本需求, 扩大生产需要的资本开支, 资本开支占利润比
- **销售过程**: 销售含现量, 应收 / 预收占比
- **运营过程**: 运营资金结构, 现金周转周期
  - 现金周转周期 = 存货周转天数 + 应收周转天数 − 应付账款周转天数
  - 越小越好, 为负数最好

数据源: 年报"现金流量表 + 附注"; 年报"资产负债表"

### §1.7 已知优秀商业模式特性

指导问题 — 对照 6 种已知优秀商业模式, 判断公司符合哪几条:

1. **知名品牌的短周期产品制造商** — 举例: 箭牌口香糖、可口可乐、吉列、斑马墨带
2. **必需品或公共事业产品的提供商** — 举例: 药品、水电等公共产品
3. **因品牌 / 形象 / 技术 / 质量之故, 产品能以溢价销售的公司** — 举例: 斯沃琪集团、路易威登、奥迪、蒂芙尼
4. **由外部影响和监管带来需求的产品的提供商** — 举例: 罗森博格国际、GEICO 保险
5. **有高度扩展性的企业, 产品的边际成本接近于零** — 举例: 微软、SAP、甲骨文、辉瑞
6. **市场上最便宜产品的供应商** — 举例: 沃尔玛、亚马逊、内布拉斯加家具城

数据源: 行业研报; 年报"核心竞争力"

## §3 护城河分析

> 好公司两个标准: 它做的事别人做不了; 它做的事自己可以重复做。

本 §3 全部 subsection 必须遵循 5 步护城河分析法:

a. 分类: 大（规模经济/网络效应）/ 准（成长期雏形, 某一维度已现苗头但 5 年以上可证伪增长尚未达标）/ 强（品牌/定价权/客户黏性）/ 省（低成本结构）/ 专（特许/牌照/不可复制工艺）
b. 2 项可证伪检验: 提价检验/对手检验/切换成本场景 / ROE 路标 四选二, 必须带具体数字。
c. 跨年定量追溯: 近 5 年毛利率/净利率/ROE 稳定性 + 经营现金流/净利润 比值, 每个数字带 (年报-YYYY.pdf p.NN)。
d. 悲观情景: 指名一个能打破该护城河的具体情景（技术切换/消费偏好/监管/对手变招）, 不写"难免有风险"这种空话。
e. 宽/中/窄/弱 标签:
   - 宽: 5 项满足 ≥ 4 项 + 关键检验 ≥ 10 年历史支撑 + 未来 10 年可预见无颠覆情景。
   - 中: 满足 3 项, 关键检验 5-10 年支撑; 未来 5 年维持, 10 年存疑。
   - 窄: 满足 2 项, 关键检验仅 3-5 年; 未来 3 年维持, 5 年存疑。
   - 弱: 仅满足 1 项, 关键检验 < 3 年, 或无定量支撑; 不建议作为买入核心论据。
   - "准"型分类上限为"中"（按定义尚未完全成型）。

禁用空话: "强大品牌"/"行业龙头"/"核心竞争力突出"/"护城河宽广"/"成长空间巨大" 等 8 条, 无具体证据一律退回重写。

### §3.1 定性: 公司的护城河有哪些

指导问题:
- 大类（规模经济/网络效应）: 公司是否在某业务上达到最小有效规模? 是否有双边/多边网络效应?
- 准类（成长期雏形）: 公司在品牌/渠道/技术某一维度是否有苗头但尚未满 5 年可证伪增长?
- 强类（品牌/定价权/客户黏性）: 能否涨价而量不掉? 客户是否占据心智?
- 省类（低成本结构）: 成本优势来自地理/工艺/采购/规模中的哪一项? 可持续吗?
- 专类（特许/牌照/不可复制工艺）: 是否有监管准入、专利、或工艺壁垒?

数据源: 年报"核心竞争力"; 招股说明书

### §3.2 护城河来源因素分析

指导问题:
- 无形资产（品牌/专利/垄断/秘方/地理独占）?
- 转换成本?
- 网络效应?
- 低成本（规模/区域/商业模式创新）?
- 有效规模/利基市场?

### §4.8 大股东与上市公司利益一致性检查

系统检查大股东是否通过多种明暗渠道抽血/掏空上市公司。即使公司为"好生意", 大股东侵占若持续存在, 会逐年侵蚀小股东权益。本节与 §4.6 "以股东利益为导向"(正面: 对所有股东的回报)形成正反对照, 与 §4.7 "股权结构"(事实: 谁持股)承接。

**与 Part 4 §4.5 排雷 29 项的区别**: §4.5 是**定量触发式**检查(财报阈值); 本节是**定性机理分析**(大股东动机 + 渠道图谱 + 历史行为模式)。两者互补, 不重复。

指导要求:
- 识别所有潜在抽血渠道, 逐项判断是否存在(是/否/不适用 + 证据)。
- 每项带年报附注页码或公开披露 URL, 无证据标 "需人工"。
- 国企(控股股东为国资委/地方政府/省国资)需特别关注政策性任务/地方采购/税收土地反哺等"合规抽血"。
- 民企(大股东为自然人或家族)需关注资金占用/担保/关联方转移。
- 即使触发项"无伤大雅", 也要记录——长期累积会侵蚀复利。

数据源: 年报附注 5 或 6 "关联方及关联交易" + 附注"其他应收款"+"预付款"+"担保"; 董事会报告"重要事项"; 历史处罚公告; 招股说明书关联交易清单; 财新/上海证券报 相关报道

**抽血渠道逐项检查清单** (10 项):

| # | 渠道 | 查看位置 | 触发条件 |
|---|---|---|---|
| 1 | 资金占用(其他应收款关联方) | 附注"其他应收款"明细 + 附注关联方 | 对大股东/关联方其他应收期末余额 > 流动资产 5%, 或长期挂账 ≥ 1 年 |
| 2 | 关联采购不公允定价 | 附注关联交易(关联采购金额 + 单价对比第三方) | 关联采购 > 营业成本 10%, 或单价显著高于第三方可比价 |
| 3 | 关联销售不公允定价 | 附注关联交易(关联销售金额 + 毛利率对比) | 关联销售 > 营收 10%, 或毛利率显著低于第三方销售 |
| 4 | 商标/品牌授权费上缴集团 | 附注无形资产授权 + 管理费明细 + 关联交易 | 大股东收取年度商标使用费 > 净利润 1%, 且定价非市场化 |
| 5 | 销售/采购渠道被大股东控制 | 前 5 大客户/供应商 + 关联方销售渠道 | 通过大股东下属子公司销售 > 营收 20%, 或采购 > 成本 20% |
| 6 | 对外担保给关联方 | 附注"重大对外担保事项" | 担保金额 > 净资产 10%, 或给大股东/关联方担保(非对子公司) |
| 7 | 预付款长期挂账关联方 | 附注预付账款对象 + 账龄 | 预付关联方同比 > 100% 或账龄 > 1 年且金额 > 流动资产 3% |
| 8 | 大股东主导合营项目资金沉淀 | 附注其他非流动资产 + 合营/联营明细 | 对大股东主导合营项目净投入 > 净资产 5% 且 ROE < WACC |
| 9 | 高管薪酬由大股东集团代付/分担 | 附注董监高薪酬 + 关联方劳务 | 上市公司付给集团的劳务费 > 营收 1% 且含高管薪酬成分 |
| 10 | 分红率大股东 vs 小股东差异 | 利润分配方案 + 控股子公司利润留存 | 大股东要求分红率 > 上市公司整体(大股东急套现), 或反之 |

**综合判定**:
- **0 触发** → 大股东无侵占, 小股东利益与大股东完全一致
- **1-2 项 低程度触发** → 轻度抽血, 不影响投资论点, 但需监控演化
- **≥ 3 项触发 或 任一"中高程度"触发** → §4.6 置信度降一档, 估值折扣加深 5-10%
- **≥ 5 项触发 或 任一"严重"触发** (如 > 10% 不公允关联交易 / 资金占款规模 > 净资产 5% / 大股东质押连环担保) → §2.12.2 **一票否决**, profile 终止

**国企特殊提示** (控股股东为国资委/地方政府/省国资):
抽血渠道可能包括以下"合规抽血"形态——不是违规但对 small shareholder 是机会成本:
- **政策性任务**: 上市公司承担非经济效益的社会责任 (扶贫/就业/维稳/地方产业配套)
- **地方政府强制采购**: 要求使用特定本地供应商, 定价高于市场
- **税收/土地优惠 vs 反哺**: 看似给优惠, 实则要求配套投资/慈善捐赠
- **商标与品牌权属**: 母公司集团持有核心商标, 上市公司每年付使用费
- **集团销售公司分成**: 销售环节通过集团下属销售公司, 吃中间差价
- **公益与基建摊派**: 地方性基建/公益项目强制配套出资

**茅台案例** (启发本节设计): 贵州茅台 的 控股股东 茅台集团 (贵州省国资委 100% 持股) 历年通过以下渠道 低强度抽血:
1. 商标 "茅台" 归 集团, 上市公司 支付年度 使用费
2. 集团销售公司分成 非上市主体 吃 加价差
3. 地方性 慈善/公益 承诺 (支持贵州 教育/扶贫/文化)
4. 省政府配套基建 (机场/水库/道路) 摊派 投资

对茅台这种超强护城河+高 ROE 生意, 这些渠道"无伤大雅"(年度合计 < 净利润 3%, 估值影响有限), **但若是差生意+同样抽血, 则小股东几乎无剩余利润**。因此: **护城河强度 > 抽血强度 是唯一救赎**。

**民企特殊提示** (大股东为自然人或家族):
- 关注关联方担保 (大股东个人企业抵押上市公司资产)
- 关注 "突击分红" (IPO 后短期内分光多年累积)
- 关注 "股权质押 + 再融资" 循环 (大股东套现)
- 关注 家族成员 任职 + 薪酬(是否任人唯亲/裙带关系)

**历史监管处罚记录**: 列出公司历史上是否有过与大股东抽血相关的监管处罚 (证监会/交易所/审计意见) — 有处罚=强信号, 降置信度两档。

- 高杠杆地产 (§6.4) — 用 三道红线/有息负债/EBITDA / 销售回款/存货分项, 替换常规 ROE / 运营
- 公用事业 (§7.4) — 用 上网电量/度电净利/股息率 / FCF/NI, §Q4 成长标 N/A (稳态)
- 资源/周期 (§5.4) — 用 穿越周期平均利润/周期 ROE 均值, §Q4 成长标 N/A
- 白酒/消费品 (§1.4 / §4.4) — 多数 §Q 适用, 按 附加检查 补行业特性

某 §Q 在本行业无意义 → 标 `N/A (原因)`, 绝不硬凑指标。

### §Q1 盈利分析

取数要求: 从年报"主要会计数据和财务指标" + "所有者权益" + "负债表"取 ROE, ROA, 净资产, 留存利润, 有息负债（短期借款 + 长期借款 + 应付债券 + 一年内到期的非流动负债）。

### §Q2 盈利能力

取数要求: 从年报"利润表" + 附注"营业成本/税金及附加/销售/管理/财务费用"取毛利率（>50% 优秀）, 核心利润率（= 毛利 − 税金附加 − 三费）, 净利率（>20% 优秀）。

### §Q3 盈利质量

取数要求: 从年报"现金流量表" + "利润表"取销售商品提供劳务收到的现金, 营业收入, 经营活动现金流净额, 净利润, 购建固定资产无形资产和其他长期资产支付的现金。计算销售收现率（>1.17）, 净现比（>1）, 资本开支占净利润比（<30% 优秀）。

### §Q4 成长能力

取数要求: 从年报"主要会计数据"取营业收入, 净利润, 归母扣非净利润; 计算营收增速/净利增速/扣非增速。

### §Q5 资产增速

取数要求: 从年报"资产负债表"取总资产, 总负债, 净资产（所有者权益合计）; 计算增速。

### §Q6 运营能力

取数要求: 从年报"资产负债表" + "利润表" + 附注取总资产, 存货, 应收账款, 应付账款, 营业成本, 营业收入。计算周转天数（= 365 / 周转率）和现金周转周期（存货 + 应收 − 应付）。

### §Q7 偿债能力

取数要求: 从年报"资产负债表" + 附注"借款"取短期借款, 长期借款, 应付债券, 一年内到期非流动负债, 货币资金, 流动资产, 流动负债, 存货。指标: 有息负债率（<20% 健康, >50% 危险）/ 有息现金覆盖率（>1）/ 资产负债率/ 流动比率（>2 安全）/ 速动比率（1.5-2）。

### §Q8 现金流分析

取数要求: 从年报"现金流量表" + "利润表" + "利润分配表"取经营活动现金流净额, 净利润, 销售商品提供劳务收到的现金, 营业收入, 货币资金, 资本支出, 现金分红总额, 有息负债余额。

子表: 8a 净利润含现量; 8b 销售收现率; 8c 现金余额 vs 资本开支 vs 分红 vs 有息负债。

### §Q9 资产负债表结构分析

取数要求: 从年报"资产负债表" + 附注取货币资金, 交易性金融资产, 固定资产 + 在建工程 + 无形资产（生产资产）, 存货, 应收账款, 预付款项, 商誉, 短期借款 + 长期借款 + 应付债券（有息负债）, 应付账款 + 预收账款（营运负债）, 所有者权益。计算各科目占总资产比。

风险指标: 商誉/应收/应付/预付占比。

### §Q10 简化利润表分析

取数要求: 从年报"利润表" + 附注"非经常性损益"取营业收入, 营业成本, 税金及附加, 销售费用, 管理费用, 研发费用, 财务费用, 资产减值, 信用减值, 投资收益, 营业外收入, 营业外支出, 所得税, 净利润, 非经常性损益。所有科目/营收 = 百分比。

### §Q11 上下游分析

取数要求: 从年报"资产负债表"取应付账款, 预收账款/合同负债, 应收账款, 预付款项。对比占总资产/占营收比; 比较上游 vs 下游议价能力。

判读: 上游话语权 — 预付低 + 应付高 = 强; 下游话语权 — 预收高 + 应收低 = 强。

### §Q12 定量分析总结

指导问题:
- 本公司在盈利/成长/运营/偿债/现金流哪几维度具有明显优势?
- 哪些维度低于行业/出现趋势恶化?
- 定量结论支持还是修正 Part 1 的定性判断?

数据源: §Q1-§Q11 的汇总

| 24 | 前五客户/供应商 >50%; 单一 >30% | 年报第四节前五大客户 + 前五大供应商披露 | 前五客户 > 50% 或 单一客户 > 30%; 前五供应商同理 | 依赖度风险; 查单一客户是否关联方 + 合同期限 + 续约历史 |
| 25 | 研发费用资本化比例/占净利润比 | 附注研发费用明细（资本化 vs 费用化） | 资本化率 > 30% 或 资本化金额 > 净利润 10% | 利润虚增嫌疑; 按全额费用化重算扣非利润 |
| 26 | 折旧/摊销会计政策变动 | 附注重要会计政策变更披露 | 当年主动变更折旧年限或残值率（非准则强制变更） | 一次性利润调节; 还原原政策重算并标注 |
| 27 | 应收账款坏账计提比例变更/保理高 | 附注坏账政策 + 保理业务披露 | 计提比例显著放松（单账龄区间减半）; 或保理金额 > 应收 30% | 利润虚增 or 将信用风险外包; 还原原坏账政策 |
| 28 | ROE <15% | 近 5 年 ROE | 5 年平均 ROE < 15%, 或任一年 < 10% | 不满足好生意门槛; 三大前提 ③ '存疑', 估值打折扣或放弃 |
| 29 | 关联交易占比高/大股东是信托或投资公司 | 附注关联方及关联交易 + 股东结构 | 关联销售 > 营收 10% 或 关联采购 > 营业成本 15%; 大股东是信托/有限合伙/投资公司（实控人不透明） | 定价公允性深查; 大股东结构不透明 → 降置信度一档 |

### §4.6 当前价位买入仓位

指导问题:
- 本标的是否在我的能力圈内?
- 容错机制（分批/止损/上限）是什么?
- 当前价位的拟建仓比例?

数据源: Part 4 §4.2 + §4.3

### §4.7 买入后加仓/持有预案

指导问题:
- 下跌 10% 的应对（加仓/持有/观望）?
- 下跌 20% 的应对?
- 下跌 30% 的应对?
- 下跌 50% 的应对（业务是否根本性恶化?）?

数据源: Part 1 综合; 个人风险偏好

### §4.8 买入前最终检查

指导问题:
- 研究是否彻底/有无遗漏?
- 有没有被我忽视的重大信息?
- 我是否愿意持有 10 年（假设不能卖）?
- 仓位是否合理?
- 完全亏损（极端情景）我能否承受?

数据源: 整体 profile 自检

### §4.9 心理偏误 核查

指导问题:
- 是否在追涨（因为近期涨得多而急于买）?
- 是否在拒买（因为跌了不舒服而回避）?
- 是否在价格锚定（非要等某个历史低点）?

数据源: 自我反思

## Part 5 — 持有阶段检查清单

> 持有阶段的任务是拒绝卖出。股权是目的, 现金是手段。

### §5.1 组合健康检查

指导问题:
- 组合透视盈利（把所持股权加总）是多少?
- 透视 ROE / 股息率是多少?
- 1 美元原则: 每投入 1 美元, 公司是否创造了 ≥1 美元的市值?

数据源: 个人组合表 + 各持仓最新年报

### §5.2 持续关注经营状况

指导问题:
- 季报关注什么（营收增速/毛利率/现金流）?
- 半年报/年报关注什么（分红/研发/资本开支）?
- 定性因素（管理层/护城河/行业）是否需要重新审视?

数据源: 季报/半年报/年报/业绩说明会

### §5.3 是否触发卖出标准

指导问题:
- 是否看错（原论点不成立）?
- 是否基本面根本性恶化?
- 是否极度高估（远超合理区间）?
- 是否出现更好标的（机会成本）?

数据源: Part 1-4 综合 + 最新季报

### §5.4 卖出动作心理偏误 核查

指导问题:
- 是否因为涨太多而急于落袋?
- 是否因为无法承受下跌而止损?
- 是否因为成本价锚定而拒绝止盈或止损?

数据源: 自我反思

### §5.5 是否触发新的买入标准

指导问题:
- 业务大超预期（量价齐升/新业务落地）?
- 出现大熊市系统性低估?
- 出现黑天鹅导致公司非基本面下跌?

数据源: 季报/公告; 宏观/行业动态
