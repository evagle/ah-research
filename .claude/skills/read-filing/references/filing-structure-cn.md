# Filing Structure CN — A 股年报/季报结构 (证监会第2号准则)

本文件覆盖 A 股上市公司 (含沪深主板/创业板/科创板/北交所) 年报和季报的法定结构与披露时限。港股结构见 `filing-structure-hk.md`。

---

## 1. A 股年报节次地图 (证监会第2号准则, 2021修订版)

A 股年报遵循《公开发行证券的公司信息披露内容与格式准则第2号——年度报告的内容与格式》强制结构。不同公司视业务有无对应内容 节次数略有差异 (无优先股/无公司债 的公司对应节会空缺或省略, 但节次编号通常保留)。

| 节次 | 标题 | 阅读重点 | 读完应回答 |
|---|---|---|---|
| 第一节 | 重要提示、目录和释义 | 董事会保证声明/审计意见类型/重大风险/释义 (特殊术语) | 审计意见类型? 管理层敢不敢签字? 有没有董事"不保证"的陈述? |
| 第二节 | 公司简介和主要财务指标 | 基本工商信息 / 5年主要数据 (营收 / NI / ROE / 资产/负债/现金流) / 会计政策/估计变更说明 | 5年主要指标趋势? |
| 第三节 | 管理层讨论与分析 (MD&A) | 经营情况概述/行业格局/业务模式/核心竞争力/分部情况/主要会计数据变动原因/未来展望 | **最后读**; 对照前面附注事实有无美化 |
| 第四节 | 公司治理 | 治理结构/独董工作/内部控制/关键岗位人员 | 有无被监管问询/内控缺陷? |
| 第五节 | 环境和社会责任 | ESG (近年权重上升) / 排污/党建 | 高污染行业看环保处罚; 其他公司快扫 |
| 第六节 | 重要事项 | **诉讼/仲裁/处罚/担保/承诺/关联交易/重大合同/重大资产重组** | 近3年履行 vs 未履行承诺? 重大诉讼金额? |
| 第七节 | 股份变动及股东情况 | 股本变动/前10名股东/控股股东情况/董监高持股 | 套现? 质押率? 控股股东占款? |
| 第八节 | 优先股相关情况 | 仅发行过优先股的公司有; 股息支付顺序/强制派发条件 | 有无优先股压顶? |
| 第九节 | 债券相关情况 | 仅公开发行公司债/企业债的公司有; 到期日/利率/兑付能力 | 债券到期集中度? 兑付风险? |
| 第十节 | 财务报告 | **合并 + 母公司 三大表 + 附注** | 核心战场。附注12项全读 (见 `statement-reading.md §3`) |
| 第十一节 | 备查文件目录 | 索引 | 找原件时用 |

**节次与"看什么"的映射速查**:
- 看 **审计** → 第一节 + 第十节审计报告
- 看 **核心利润来源** → 第十节 (合并利润表 + 附注营收/成本拆分)
- 看 **造假风险** → 第十节附注 + 第六节重要事项
- 看 **管理层诚信** → 第四节公司治理 + 第六节承诺履行 + 第七节股东变动
- 看 **关联交易** → 第六节 + 第十节附注
- 看 **研发/资本开支** → 第十节附注 (在建工程/开发支出/研发费用/无形资产)
- 看 **分红能力** → 第十节母公司报表 (未分配利润 + 资本公积)
- 看 **偿债压力** → 第九节 (公司债到期结构) + 第十节附注 (有息负债)
- 看 **管理层承诺 vs 兑现** → 第三节"未来展望" (t 年) vs 第三节"经营情况" (t+1年) 跨年对照

## 2. 季报 vs 年报 (A 股)

A 股季报/半年报与年报的**披露深度、审计强度、法规细节**都不同。读季报犯年报那套深读逻辑的错会浪费时间; 反之用季报的浅读逻辑读年报会漏风险。

### 2.1披露时限 (强制)

| 报告类型 | 披露截止 | 审计要求 |
|---|---|---|
| 年报 | 次年 **4月30日** 前 | 年度审计 (标准无保留/保留/无法表示/否定) |
| 一季报 | 当年 **4月30日** 前 (不早于上年度年报) | 无审计 |
| 半年报 | **8月31日** 前 | 上交所/深交所 要求有限审阅 (非完整审计) |
| 三季报 | **10月31日** 前 | 无审计 |
| 业绩预告 | 见下表 | 无审计 |
| 业绩快报 | 原则上2月底前 (年度业绩快报) | 无审计 |

**业绩预告触发条件** (符合任一即必须发预告):

| 触发条件 | 上交所 | 深交所 主板 | 深交所 创业板 |
|---|---|---|---|
| 净利为负 (亏损) | 必须预告 | 必须预告 | 必须预告 |
| 扭亏为盈 | 必须预告 | 必须预告 | 必须预告 |
| 净利同比 ±50% 以上 | 必须预告 | 必须预告 | 必须预告 |
| 期末净资产为负 | 必须预告 | 必须预告 | 必须预告 |
| 年度营收 < 1亿 (退市风险) | 必须预告 | 必须预告 | 必须预告 (≤ 3亿 创业板) |

**业绩预告披露时点**:

| 报告期 | 披露期限 |
|---|---|
| 年度 | 次年 **1月31日** 前 |
| 一季度 | 当年 **4月15日** 前 |
| 半年度 | 当年 **7月15日** 前 |
| 三季度 | 当年 **10月15日** 前 |

**未按时披露/延期披露** = **强 red flag**, 进入 ST 候选/监管关注。遇此情形 → 立即早退 (read-filing SKILL.md §2.1.1 L3)。

### 2.2季报披露内容 (大幅精简)

季报 vs 年报 内容差异:

| 内容 | 年报 | 季报 |
|---|---|---|
| 三大表 (合并 + 母公司) | 有 | 有 (通常未审) |
| 附注 | **详细12项 + 全部政策** | 仅**主要会计数据变动说明**, 通常 < 5页 |
| 董事会报告 / MD&A | 有 | 有但大幅精简 |
| 重大事项 | 有 | 有 (重大合同/诉讼/关联交易/承诺履行) |
| 股东人数/前10股东 | 有 | 有 |
| 审计报告 | 有 | **无** (半年报有审阅报告) |
| 公司治理/内控 | 有 | 无 |
| 行业分析 | 有 | 无 |

**操作推论**: 季报阅读时间 ~1小时/家 (年报 ~3-4小时)。附注深读在季报模式下简化为"主要会计数据变动说明 + 期末三大表 结构"。

### 2.3季报必看4点 (YoY + QoQ + 累计)

1. **累计 YoY**: Q1 = 当季 = 累计; Q2 = 半年累计; Q3 = 前三季累计。累计数据对比去年同期同范围
2. **当季 QoQ**: 算出当季单季数据 (Q3单季 = 三季累计 − 半年累计), 看环比。**QoQ 比 YoY 更早发现经营拐点**
3. **CFO 累计 / NI 累计**: 季报时点计算, 连续3个季度 < 50% → 升级关注
4. **合同负债/应收/存货/预付** 期末结构: 环比上季末变化 > ±20% 必须有解释 (年报附注级数据在季报里未必详细, 但主要科目变化 + 变动原因在季报"主要会计数据变动说明"里应披露)

### 2.4季报专属 red flag

- **季报披露延期** (§2.1)
- **重述季报** (季度发完后再补发修正)
- **季报 vs 业绩预告偏离 > ±20%** (预告 "预计净利 增长30-50%", 实际披露5%)
- **Q4单季 断崖** (Q4 = 年报累计 − 三季累计; 若 Q4单季 << Q1-Q3季度均值, 可能是年末审计调整/一次性坏账)
- **Q3业绩大幅低于预期但 Q4突击反转** (年底突击调利润的典型)

### 2.5 Q3 = 距离年报最近的高价值季报

Q3报表披露9个月累计, 距离年报仅剩1个季度 (约3个月)。Q3是年报预测的**最后校准点**:
- Q3 CFO 累计 / NI 累计 → 年度比率的可靠估计
- Q3应收/存货/合同负债 → 年底水位预估
- Q3管理层口径 → 为年报埋伏笔

遇"公司业绩超预期"消息先回查 Q3数据 vs 业绩预告/当年 Q1-Q3趋势, **不要等年报再判断**。

## 3. A 股特有事项

- **会计准则**: 2007+ 切到"企业会计准则" (与 IFRS 趋同); 2018-2021完成新金融工具/新收入/新租赁 三项切换
- **税率基准**: 企业所得税法定税率25%, 高新技术企业/西部大开发15%, 小微5-10%
- **实际披露平台**:
  - **巨潮资讯 cninfo** `http://www.cninfo.com.cn` — 覆盖 A 股全市场
  - **上交所** `http://www.sse.com.cn` (6开头的沪市)
  - **深交所** `http://www.szse.cn` (0 / 3开头的深市/创业板)
  - **北交所** `http://www.bse.cn` (4 / 8开头)
- **语言**: 强制中文, 可选英文版 (大型 H 股可能发双语)
