# Filing Structure HK — 港股年报 / 中报结构 (HKEX Main Board Listing Rules Appendix 16)

本文件覆盖香港交易所主板 (Main Board) 上市公司的年报和中报结构。GEM 板 (创业板) 结构类似但披露时限略紧。A 股结构见 `filing-structure-cn.md`。

---

## 1. 港股年报结构 (HKEX Appendix 16 + Cap. 622 公司条例)

港股年报不像 A 股有"第 N 节"硬编号, 而是按**章节标题**组织。大小节顺序由公司决定, 但**必须包含以下内容** (HKEX Rule 13 + Appendix 16 + 公司条例 Schedule 5):

| 章节 | 英文原名 | 内容 | 阅读重点 | A 股对应 |
|---|---|---|---|---|
| 公司资料 | Corporate Information | 董事名单 / 公司秘书 / 注册办事处 / 主要银行 / 审计师 / 法律顾问 / 股份登记处 | 审计师 / 公司秘书 变更史 | 第一节封面 |
| 主席报告 | Chairman's Statement | 战略方向 / 业绩简评 / 展望 | 管理层承诺兑现的主要依据 | 第三节 MD&A 部分 |
| 管理层讨论与分析 | Management Discussion and Analysis (MD&A) | 业务回顾 / 财务回顾 / 分部情况 / 资本结构 / 流动性 / 外汇风险 | **对照附注实际数据** | 第三节 MD&A |
| 董事及高管简介 | Directors' and Senior Management's Biographical Details | 任职年限 / 专业背景 / 其他上市公司职务 | 独立董事真独立性 / 关联方 | 第四节 公司治理 |
| 董事会报告 | Directors' Report (法定, Cap. 622 Schedule 5 强制) | 主要业务 / 财务结果摘要 / 分红建议 / 储备变动 / 捐款 / 五年财务摘要 / 董事持股 / 购回股份 / 主要客户和供应商集中度 / 关联交易 | **5 大客户 / 供应商 占比 — A 股年报无此披露, 港股强制** | 第三节 + 第六节 |
| 企业管治报告 | Corporate Governance Report (CG Code 强制) | 治理架构 / 董事会会议出席率 / 薪酬委员会 / 审核委员会 / 风险管理 / 内部控制检讨 / ESG 治理 | 独立审核委员会对财报的保证声明 | 第四节 |
| ESG 报告 | Environmental, Social and Governance Report (Rule 13.91, 2016 强制) | 环境 KPI / 员工 / 供应链 / 社区 | 高污染行业重点; 其他公司 快扫 | 第五节 |
| 独立核数师报告 | Independent Auditor's Report | 意见段 + 关键审计事项 (Key Audit Matters, KAM) | **KAM 是港股审计披露的金矿**, 直接说审计师关注的风险点 | 审计报告 |
| 综合财务报表 | Consolidated Financial Statements | 综合损益表 / 综合全面收益表 / 综合财务状况表 / 综合现金流量表 / 综合权益变动表 | 三大表主体 | 第十节 |
| 财务报表附注 | Notes to the Consolidated Financial Statements | 会计政策 / 附注 1-N | **12 项必读附注** (见 `statement-reading.md §3`) | 第十节附注 |
| 五年财务摘要 | Five-Year Financial Summary | 5 年关键财务数据 | 纵向趋势 快扫 | 第二节 |

**港股 vs A 股 结构差异的要点**:

1. **没有"监事会报告"** — 港股是单层董事会 + 独立非执行董事 (INED); 不设监事会。A 股惯用的"监事会偶尔藏信号"这条经验对港股无效。
2. **没有"合并 + 母公司" 两套报表** — 港股只披露**综合** (consolidated) 报表, 不强制单独披露母公司报表。母公司数据仅在年报底部的"公司财务状况表" (Statement of Financial Position of the Company) 简要给出, 用于判断分红来源。
3. **董事会报告里强制披露前 5 大客户 + 前 5 大供应商 占比** — A 股通常只在附注披露前 5 大客户应收账款, 不强制披露营收集中度。港股 5 大客户 营收占比 > 30% 必须披露, 占比 > 50% 是显著红旗。
4. **KAM 披露更实质** — 港股审计师必须列出 Key Audit Matters 且解释怎么核的, 比 A 股审计报告更具体。
5. **ESG 强制披露** — 2016 年起, HKEX Rule 13.91; A 股 2024+ 在沪深交易所头部公司强制, 覆盖面晚于港股。
6. **关联交易** 按 HKEX Chapter 14A 披露, 不同于 A 股第 2 号准则。

## 2. 港股中报 / 季报结构

### 2.1 披露时限

| 报告类型 | 披露截止 | 审计要求 |
|---|---|---|
| 年报 (Annual Report) | 财年结束后 **4 个月内** (e.g. 12-31 年结 → 4 月 30 日) | 法定全面审计 (Cap. 405 核数师条例) |
| 年度业绩初步公告 (Preliminary Results Announcement) | 财年结束后 **3 个月内** | 与审计师商定 (通常审计已完成) |
| 中期报告 (Interim Report) | 上半年结束后 **3 个月内** (e.g. 06-30 → 9 月 30 日) | 可选 (大部分公司选"审阅" review, 非完整审计) |
| 中期业绩初步公告 | 上半年结束后 **2 个月内** | 通常无审计 |
| 季报 (Quarterly Report) | **不强制** (主板); GEM 板 强制, 45 日内 | 无审计 |

**重要差异**:
- **主板无季报义务**, 只有年报 + 中报 两次定期披露。所以港股定性研究节奏是"年报深读 + 中报复核"两次, 而非 A 股的"四次定期披露"。
- **GEM (创业板) 有季报**, 45 天内必须披露 Q1 / Q3。0700.HK 等主板公司不适用。
- **Profit Alert / Profit Warning** (HKEX Rule 13.09): 公司一旦察觉业绩重大变化 (通常 ± 30-50%) 必须**即时发出盈利预警 / 盈利警告**, 时点不由定期报告日期决定。

### 2.2 中报披露内容

中报 vs 年报:

| 内容 | 年报 | 中报 |
|---|---|---|
| 综合财务报表 | 有 (法定审计) | 有 (通常 review) |
| 附注 | **详细** | **简化版, 只披露重大变动** |
| MD&A | 有 | 有 (通常简短) |
| 董事会报告 | 有 (法定) | **不要求** |
| 企业管治报告 | 有 (法定) | 不要求 (重大变更会单独 announce) |
| ESG 报告 | 有 (法定, 可独立成册) | 不要求 |
| 审计报告 | 标准无保留 / 保留 / 否定 | 通常无, 有时披露 review 结论 |
| 关联交易 | 全年汇总 | 半年汇总 |

**操作推论**:
- 港股中报 (上半年业绩) 是年报之外**唯一**有定量数据的官方披露, 价值高于 A 股半年报
- 中报读法: 对照上年年报的 MD&A 预期 → 复核核心指标 (营收 / 毛利率 / 净利率 / CFO) → 检查分部结构 是否按预期发展

### 2.3 港股专属 red flag (与 A 股差异)

- **Profit warning 突然发布** (尤其离中报 / 年报披露日 < 1 个月) = 强信号
- **审计师辞任 / 变更** — 港股审计师变更有独立公告, 核数师辞职信必读 (如果发出)
- **关联交易超阈值无独立股东批准** (HKEX Chapter 14A) — 需要查公告
- **核数师保留意见 / 保留事项** — 港股审计师保留事项写得比 A 股更具体, 读 KAM 段
- **大股东股份质押比例 > 50%** 及附近波动 — HKEX Rule 13.18 强制披露大股东质押

## 3. 港股特有事项

- **会计准则**: HKFRS (与 IFRS 完全趋同, 几乎零差异)
- **语言**: 英文或中文均可, 大部分公司发双语。**研究优先读英文版** (避免翻译歧义, 尤其专业术语)
- **披露平台**: **HKEX 披露易** (`https://www.hkexnews.hk`) — 覆盖港股全市场
- **财年**: 大部分公司 12-31 年结; 部分 06-30 / 03-31 (如长和系)。读前先查公司财年
- **税率基准**: 香港利得税法定 16.5%; 两级制首 200 万港元 8.25%; 离岸所得免税。**这个远低于 A 股 25%**, 同一财务指标解读可能不同 (e.g. "所得税 / 税前利润 20%" 在 A 股是 red flag, 在港股是正常)
- **A+H 双重上市** (如工行 / 中石油): 两本年报内容一致但格式不同, 港股年报更简洁 (无监事会报告 / 无党建 / 无 ESG 中文党政话术)
- **H 股 vs 红筹** 结构差异: H 股 = 在内地注册, 在香港上市; 红筹 = 在境外 (常开曼) 注册, 在香港上市。两者年报结构大致相同, 但红筹公司可能有更复杂的集团架构 (VIE 结构特别需要警惕)

## 4. 港股 / A 股 审计与治理的实务差异

| 维度 | A 股 | 港股 |
|---|---|---|
| 董事会结构 | 董事会 + 监事会 (双层) | 单层董事会 + 独立非执行董事 (INED) |
| 审计师变更披露 | 年报"审计机构情况" 段落 | 独立公告 (HKEX) |
| 审计意见类型 | 标准无保留 / 保留 / 无法表示 / 否定 | 同, 但措辞更详尽, KAM 是标配 |
| 关联交易 监管 | 证监会 + 交易所自律规则 | HKEX Chapter 14A, 阈值披露规则详细 |
| 大股东占款披露 | A 股历史上的突出问题, 年报附注"其他应收款"关联方行 | 港股较少见 (公司法严格), 但红筹公司仍需警惕 |
| 股份回购 | 强制注销或库存 | 可注销 / 库存 / 托管; 披露更灵活 |
