## 1. 术语快查 (防混淆)

| 易混 A | 易混 B | 关键区别 |
|---|---|---|
| 母公司报表 | 合并报表 | 前者只上市公司本部; 后者含全部子公司 |
| 少数股东权益 | 少数股东损益 | 前者在资产负债表 (净资产口径); 后者在利润表 (净利口径) |
| 归母净利 | 合并净利 | 前者 = 后者 − 少数股东损益 |
| 货币资金 | 现金及现金等价物 | 前者含受限; 后者 = 货币资金 (扣受限) + 短期高流动性投资 |
| 营业收入 | 营业总收入 | 部分行业 (银行) 有差异; 工业企业两者基本相同 |
| 扣非净利 | 净利 | 扣非 = 净利 − 非经常性损益 |
| 应收票据 | 应收账款 | 前者有票据背书 (银票/商票); 后者无票据 |
| 合同负债 | 预收账款 | 新收入准则下 大部分 预收账款 重分类到 合同负债; 意义 = "客户预付的还没交付的钱" |
| 经营活动 vs 投资活动现金流 | — | 定义随行业变化; 银行借款是经营, 其他公司借款是筹资 |
| 公允价值变动损益 | 投资收益 | 前者是未实现持有期变动; 后者是实现的买卖差价/利息/分红 |

## 6. 新准则3项对 BS 结构的影响 (2018+ 切换)

| 新准则 | 生效期 | BS 结构影响 | 阅读提示 |
|---|---|---|---|
| 新金融工具准则 (IFRS 9对标) | 2018-2020 | 金融资产4分类 (摊余成本/笋子/菜粽/肉粽) 替代原 "4分类 + 可供出售"; 预期信用损失模型 (ECL) 替代"已发生损失" | 笋子占比大 = 利润波动来自估值 |
| 新收入准则 (IFRS 15对标) | 2020全面施行 | 大部分原 "预收账款" 重分类到 **合同负债**; 履约进度改"按时点/按时段"确认 | 比较期数据可能追溯重述, 5年对比需注明; 合同负债/营收 同业对比 |
| 新租赁准则 (IFRS 16对标) | 2021全面施行 | 原表外经营租赁改入表: 借方 **使用权资产** + 贷方 **租赁负债**; 折旧增加, 经营租赁费用消失, 利息费用增加 | 影响重资产租赁密集型 (零售/航空/物流) 的资产负债率 / EBIT / CFO |

## 7. CFO / CFI / CFF 8组合画像 (快速定性)

| 组合 | CFO | CFI | CFF | 典型画像 |
|---|---|---|---|---|
| A | + | − | − | **成熟优质**: 自身造血 → 投资扩张 + 还债分红。典型蓝筹 |
| B | + | − | + | **扩张期**: 造血 + 举债 一起上, 激进扩张 |
| C | + | + | − | **收缩/转型期**: 主业仍造血, 但处置资产 + 还债 |
| D | + | + | + | **囤现金**: 造血 + 处置 + 举债 三方流入 |
| E | − | − | + | **初创/激进投入期**: 主业不造血, 靠融资扩张 |
| F | − | + | − | **衰退清算**: 卖资产还债, 走向清算 |
| G | − | + | + | **困境续命**: 卖家当 + 借钱 硬撑, 极度危险 |
| H | − | − | − | **极度恶化**: 三头流出, 现金耗尽, 破产前夕 |

5年轨迹画一遍, 组合突变 (A→E 或 B→G) 比单期 NI 变化更早预警。

## 8. 所有者利润 速算

```
所有者利润 ≈ CFO − 维持性 CapEx
维持性 CapEx 估算:
  法 A (折旧法)  维持性 CapEx ≈ 折旧摊销 × (1+通胀系数)
  法 B (产能法)  把历史 CapEx 拆成"新产能对应部分"与"替换部分", 后者 = 维持性
  法 C (下限法)  长期 CFO > 全 CapEx → 维持性 < 折旧
用它算 PE:
  市值 / 所有者利润 ← 报给决策用
  市值 / 归母 NI ← 作对比参考
  两 PE 差距 > 30% → 说明维持性 CapEx 估算或会计 D&A 与实际经济折旧差距大, 用"所有者利润 PE" 做主锚
```

## 9. 常见错误自检 (子 agent 提交前跑一遍)

| 自检项 | 通过条件 |
|---|---|
| 所有数字都带页码 | grep 笔记, 每个数字附近有 `(年报-YYYY.pdf p.N)` 或等价格式 |
| 附注12项100% 覆盖 | 笔记 §附注 section 有12个 entry (未披露也要写"未披露 p.N") |
| 8条禁用空话0命中 | grep 8条空话, 仅允许出现在 `>` 引用块内 (管理层原话) |
| 净利润相关 finding 都有 CFO/NI 背景 | 每条带"净利/营收/毛利率" 的 finding 附近有同期 CFO/NI/销售收现 |
| 口径一致 | 全篇"净利"默认归母, 不默认合并 |
| 5年纵向 + 3家横向 成对出现 | 核心指标表格有5年列 + 同业3列 |
| 变动 > ±20% 都已归因 | 每个大变动3句解释 (原因/证据/可持续性) |
| 审计意见类型写了 | 笔记开头含审计意见类型 + 会计师事务所 + 近5年变更次数 |
| 会计政策/估计变更 单列 | 有专章 (无变更也要明确写"本年报无变更") |
| 早退检查4项跑过 | 开头 `早退触发:` 字段非空 ("无" 也算) |

## 10. 反模式 (Bad practices) 20条

读到这里的子 agent 把它们当 red flag 自检, 主 agent 复核时也按此过滤。

| # | 反模式 | 正确做法 |
|---|---|---|
| 1 | 只看利润不看现金流 | 三表联动, §2.3.1强制 CFO / NI 背景 |
| 2 | 只看正文不看附注 | §1.5 + §2.2阅读顺序 + §2.5.1 12项100% 覆盖 |
| 3 | 只看合报不看母报 | §2.4.2母报3个使用场景 |
| 4 | 只看当年不看5年趋势 | §2.6.1 5年纵向强制 |
| 5 | 读管理层口径不交叉验证 | §2.2 step 9管理层讨论对照 + §2.10承诺 vs 兑现 表 |
| 6 | 相信评级/目标价/分析师预测 | 研报只取事实不取观点 (见 value-profile §2.11.5) |
| 7 | 相信业绩预告精度过高 | 业绩预告是区间披露, 常 ±20%; 算 PE 用实际年报 |
| 8 | 混淆少数股东权益 vs 损益 | §1.4严格区分 + §2.4.3变动30% 解释 |
| 9 | 用未扣非净利算 PE | §2.3.3强制扣非 |
| 10 | 跳过审计报告和会计政策变更 | §2.1.1 L1审计红线早退 + §2.5.2会计变更单列 |
| 11 | 把 "自由现金流 = CFO − 全 CapEx" 不拆维持性/扩张性 | §1.9所有者利润 + §2.9双口径 |
| 12 | 把 CFO ≈ 营收 × 常数 | 三表勾稽 §2.3.1强制, 不套公式 |
| 13 | 跨行业套同一个护城河模板 | 行业性差异看 value-profile industry-overlays.md |
| 14 | 扣非当作唯一利润质量指标 | 扣非 + CFO / NI + 销售收现 + cash conversion + ROOCE 五位一体 §2.3.4 |
| 15 | 季报看当季不看累计 + YoY | `filing-structure-cn.md` §2.3季报读法 |
| 16 | 把"一次性"减值/重组/诉讼 当真 | §2.3.5 5年累积 ≥ 3次 = 经营性成本 |
| 17 | 只读当年"未来展望"不回溯 | §2.10.1 forecast vs actual 5年表 |
| 18 | 忽视所有者权益变动表 | §2.5.6股权激励/回购注销 / OCI 必看 |
| 19 | 用"EBITDA × 倍数" 估值 | D&A 是真实成本; EBITDA 掩盖 CapEx → 一律拒用, 用所有者利润 |
| 20 | "财务报告" 200+ 页读不完就跳 | 附注就是价值所在; 读不完不下注, 不代表可以跳过 |
