# Statement Reading — 三大表 + 附注阅读操作手册

本文件是 `read-filing` skill 的深度操作参考, 专管 **"怎么读三大表 + 附注 + 其他章节"**。风险触发判定不在本文件管, 由 `financial-redflag-scan/references/fraud-library.md` 覆盖。

---

## 1. 阅读顺序

年报500+ 页, 按下列顺序读, 发现风险就立即止步:

1. **封面 + 目录** — 审计机构/是否变更/报告期
2. **重要提示 + 风险因素** — 管理层承认的问题在哪
3. **董事会报告** — 管理层对年度经营的自评
4. **经营业绩讨论** — 产品线/区域/渠道分解, 与年初指引对照
5. **资产负债表** — 结构是否异常
6. **现金流量表** — 利润是否变成现金
7. **利润表** — 收入结构、费用率、净利率
8. **附注** — 风险的真正藏身处
9. **审计报告** — 意见类型

**为什么先读资产负债表**: 它是利润表和现金流的 "输出累积", 且作假难度大（每笔科目都要交叉钩稽）。利润表最容易作假, 因此放在最后对照读。

## 2. 资产负债表 — 5分类 + 金融资产新准则

### 2.1五分法
- **生产经营资产**: 固定资产/无形资产/在建工程/存货 — 赚钱的生产力
- **投资资产**: 长期股权投资/金融资产/投资性房地产 — 赚其他钱
- **经营周转资产**: 货币资金（含受限）/ 应收账款/预付账款 — 业务周转
- **应付经营资产**: 应付账款/预收款项（含合同负债）/ 应付工资 — 别人欠你的
- **融资相关**: 有息负债/股东权益 — 资本结构

### 2.2金融资产新准则（2018-2020切换后）
- **以摊余成本计量** — 俗称 "瘫子", 不入当期损益
- **以公允价值计量且变动计入当期损益** — 俗称 "笋子", **直接吃利润表**, 最危险
- **以公允价值计量且变动计入其他综合收益（债权类）** — "菜粽", 入 OCI 可转出
- **以公允价值计量且变动计入其他综合收益（权益类指定）** — "肉粽", 入 OCI 终生不可转出

**读法**: 凡是 "笋子" 金额大的, 利润波动主要来自估值变动而非经营, 降级处理。

### 2.3资产负债表反直觉规则 (护城河视角)

教科书规则（Graham 式 current ratio > 2 / 低 AR 好/低负债好）对**大多数**公司适用; 但 护城河公司的资产负债表长相不一样, 用常规规则会**误判**。以下3条反直觉规则必须同时掌握:

| 指标 | 常规教科书 | 护城河公司 | 原因 |
|---|---|---|---|
| **流动比率 Current Ratio** | > 2为健康; < 1为短期偿付风险 | **< 1对 护城河公司反而正常**, 例: 可口可乐 / Moody's 长期 < 1 | 护城河公司经营现金流充沛 + 应付账款可长期占用, 不需要流动性缓冲; 真风险是经营现金流断裂, 而不是短期资产/负债比例 |
| **应收账款/营收** | 关注绝对水平 | **低于同业30%+ = 议价力信号** | 客户愿意先付款或短账期, 反映品牌/渠道/产品议价力; 高于同业 = 靠放账撑收入, 需警惕 |
| **长期有息负债** | 低负债好 | **长期负债 < 3-4年 NI 可偿清 = 护城河; < 1年可偿清 = 顶级 (可乐 / Moody's 级)** | 护城河公司不需要借钱扩张; 借了也能用经营现金迅速还完。> 5年 NI 才能还清 = 激进杠杆, 要找补什么 |

**操作要点**: 评判 "负债是否过高"、"应收是否太多"、"流动性是否紧张" 时, 必须先定性公司是 护城河还是 commodity; 同一个数字在两类公司的解读截然相反。

## 2.5利润表八大指标（护城河 / commodity 三档）

读利润表时必查以下8项, 给每项打 **护城河/中间 / commodity** 三档标签。连续10年 ≥ 6项处于 护城河档 → 护城河信号强烈; 连续2年 ≥ 3项跌入 commodity 档 → 护城河瓦解 early warning。

| 指标 | 护城河档 | 中间档 | commodity 档 | 解读 |
|---|---|---|---|---|
| **毛利率** | ≥ 40% 持续10y (顶级 ≥ 60%) | 20-40% | ≤ 20% | 品牌/成本/技术壁垒的直接量化 |
| **SG&A / 毛利** | ≤ 30% | 30-80% | ≥ 80% | 销管费吃掉多少护城河; 高比例 = 价格被迫打折/销售靠烧钱 |
| **R&D / 毛利** | ≤ 10% (消费), **≤ 25%** (科技) | 10-25% | ≥ 25% 持续 | 高 R&D 压力 = 必须持续研发才能保命的生意 (半导体/创新药), **护城河周期短** |
| **折旧/毛利** | 6-8% (轻资产) | 8-22% | ≥ 22% | 高折旧 = commodity; 护城河公司设备投入少, 折旧负担小 |
| **利息/营业利润** | < 15% | 15-49% | ≥ 49% | > 49% 濒临破产 (Goodyear / Bear Stearns 模式); 低利息 = 不依赖杠杆 |
| **所得税/税前利润** | ≈ 25% (A 股/港股实际税率) | 15-25% | < 15% 或 > 30% | **明显低于25% = 避税或优惠, 利润不可复用**; 明显高于25% 罕见, 可能附加特别税 |
| **净利率** | ≥ 20% | 10-20% | < 10% | 综合反映护城河 + 费用控制 |
| **10y EPS 趋势** | 单调上行, 最多1年小幅下滑 | 有2-3年下滑但整体上行 | 多次大幅波动/下行 | 多年稳定向上 = 护城河持久 |

**跨指标互证**:
- 高毛利率 **+** 低 Capex/NI（见 moat-framework.md §3.5）= 真护城河
- 高毛利率 **+** 高 Capex/NI = 假护城河 (折旧不充分 或 快过时的壁垒)
- 高净利率 **+** 高 R&D/毛利 = 技术护城河, 但需每年再投入 (周期性淘汰风险)
- 低 SG&A/毛利 **+** 低 AR/营收 = 议价 + 成本双优势 (可乐/茅台级)

## 3. 必读附注 (高优先级12项)

每份阅读笔记的 §附注 section 强制覆盖以下附注。遗漏 ≥ 3项视为不合格, 退回重读。**风险阈值**列同时作为 `financial-redflag-scan` 29项排雷的输入条件; 读到风险不在本文件做"触发与否"的最终判定, 交给 fraud-library.md 负责。

| # | 附注 | 看什么 | 风险阈值（作 redflag-scan 输入）|
|---|---|---|---|
| 1 | 货币资金 | 受限金额/保证金/冻结 | 受限占比 > 30% |
| 2 | 应收账款 | 5大客户名 + 账龄结构 + 坏账计提 | 单一客户 > 30%, 或1年以上占比 > 20% |
| 3 | 应收票据 | 银票 vs 商票比例 | 商票占比 > 50%（收款质量差） |
| 4 | 预付账款 | 对象 + 用途 | 对关联方预付大额 |
| 5 | 其他应收款 | 关联方占比 | 关联方占比 > 50% |
| 6 | 存货 | 分项（原材料/在产品/库存商品）+ 跌价准备 | 库存商品增速 > 营收增速 ≥ 20pp |
| 7 | 在建工程 | 项目/年限/转固计划 | > 3年仍未转固, 或占比 > 总资产20% |
| 8 | 商誉 | 并购对象/减值测试假设 | 商誉/净资产 > 30% |
| 9 | 合同负债 | 占营收比 + 同业对标 | 低于行业均值50% 可疑, 高出2× 可能提前确认收入 |
| 10 | 应付账款 | 议价权信号（账期 vs 同业） | 账期明显短于行业均值 |
| 11 | 长投 + 其他权益投资 | 被投单位有无造假/经营异常 | 被投亏损但账面不减值 |
| 12 | 有息负债 | 结构/利率/到期日 | "大存大贷"（同时有巨额现金和巨额有息负债） |

## 4. 读年报顺序（快速扫描模式）

当3年内要读 ≥ 10家公司时, 使用此快速扫描:

1. **封面 (1 min)** — 审计机构/意见/是否变更
2. **董事会报告 (5 min)** — 管理层自评 + 下年度计划
3. **重要事项 (5 min)** — 诉讼/处罚/担保/关联交易
4. **财务摘要 (5 min)** — 3-5年关键指标 (营收/利润 / ROE / CFO)
5. **附注中的12项 (30 min)** — §3的表
6. **管理层薪酬 + 持股变动 (5 min)** — 是否套现
7. **关联交易披露 (10 min)** — 金额/定价公允性

总计 ~1 hour / 家。若任一步发现风险, 该公司立即进入 "排除" 清单, 不再继续。

## 5. 对 skill 运行的约束

- 每个财报引用必须带 **年报页码**。"管理层讨论说盈利能力强" 无页码 → 退回。
- 数字引用格式: `(2024 年报 p.XX)` 或 `(招股书 p.XX)`。
- 跨年对比必须连续, 缺一年 → 说明原因（e.g. 2020年 IPO 前数据不可得）。
- 同一事项3个以上来源交叉验证（年报 + 公告 + 券商研报 + 监管披露）。
- 禁止用 "据了解/据分析师/据说" 作为财报事实来源。

## 6. 特殊场景的加读清单

| 场景 | 必加读的附注或章节 |
|---|---|
| 首次覆盖 | 近3年所有重大合同 + 管理层历史 + 股东背景 |
| 近1年变更审计机构 | 变更公告 + 前任审计师辞任声明 + 新任意见类型 |
| 商誉占比 > 20% | 每项并购的减值测试假设（折现率/永续增长率）|
| 有息负债/净资产 > 1 | 到期结构 + 再融资计划 + 利息保障倍数 |
| 金融资产 > 营业收入 | 笋子/菜粽/肉粽 分类 + 历史波动 |
| 行业有政策风险 | 法规引用 + 管理层应对 + 同业表态 |
